# 1.7 JavaScript 基础交互

JavaScript 让网页可以响应用户操作，比如点击按钮、修改文字、切换样式、播放动画。

## 引入 JavaScript

写在 HTML 中：

```html
<script>
  console.log("Hello JavaScript");
</script>
```

外部文件：

```html
<script src="js/main.js"></script>
```

通常把 `script` 放在 `body` 结束前：

```html
</body>
```

之前。

## 变量

```javascript
const name = "Alice";
let count = 0;
```

- `const` 用于不会重新赋值的变量
- `let` 用于会变化的变量

## 输出

```javascript
console.log("Hello");
```

打开浏览器开发者工具的 Console 可以看到输出。

## 获取元素

HTML：

```html
<h1 id="title">旧标题</h1>
```

JavaScript：

```javascript
const title = document.querySelector("#title");
console.log(title.textContent);
```

## 修改内容

```javascript
const title = document.querySelector("#title");
title.textContent = "新标题";
```

## 点击事件

HTML：

```html
<button id="btn">点我</button>
<p id="message">等待点击</p>
```

JavaScript：

```javascript
const btn = document.querySelector("#btn");
const message = document.querySelector("#message");

btn.addEventListener("click", function () {
  message.textContent = "按钮被点击了";
});
```

## 切换 class

HTML：

```html
<button id="btn">切换</button>
<div id="box" class="box">内容</div>
```

CSS：

```css
.box {
  background: #eee;
}

.active {
  background: #ffd54f;
}
```

JavaScript：

```javascript
const btn = document.querySelector("#btn");
const box = document.querySelector("#box");

btn.addEventListener("click", function () {
  box.classList.toggle("active");
});
```

## 常见错误

### 元素还没加载就获取

如果脚本放在 `head` 里，可能 HTML 元素还没出现，JavaScript 就执行了。

解决方式：

- 把 `<script>` 放在 `body` 结束前
- 或使用 `DOMContentLoaded`

```javascript
document.addEventListener("DOMContentLoaded", function () {
  const title = document.querySelector("#title");
  console.log(title.textContent);
});
```

### 选择器写错

```javascript
document.querySelector("title")
document.querySelector("#title")
document.querySelector(".title")
```

- `title` 选择标签
- `#title` 选择 id
- `.title` 选择 class
