# 1.3 链接、图片、列表与表格

这一章学习网页里最常见的内容组织方式。

## 链接

```html
<a href="https://github.com">打开 GitHub</a>
```

新窗口打开：

```html
<a href="https://github.com" target="_blank" rel="noopener noreferrer">打开 GitHub</a>
```

`rel="noopener noreferrer"` 可以减少新窗口带来的安全风险。

## 相对路径和绝对路径

绝对链接：

```html
<a href="https://example.com/about.html">关于我们</a>
```

相对链接：

```html
<a href="about.html">关于我们</a>
<a href="./pages/about.html">关于我们</a>
```

项目内页面和图片通常推荐使用相对路径。

## 图片

```html
<img src="images/photo.jpg" alt="一张风景照片">
```

- `src` 图片地址
- `alt` 图片说明

`alt` 很重要：图片加载失败时会显示它，屏幕阅读器也会读它。

## 无序列表

```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橘子</li>
</ul>
```

## 有序列表

```html
<ol>
  <li>打开编辑器</li>
  <li>新建 index.html</li>
  <li>写入代码</li>
</ol>
```

## 表格

```html
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>作用</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>HTML</td>
      <td>页面结构</td>
    </tr>
    <tr>
      <td>CSS</td>
      <td>页面样式</td>
    </tr>
  </tbody>
</table>
```

表格适合展示真正的表格数据，不建议用表格做页面布局。

## 常见错误

### 图片路径错误

如果图片不显示，先检查：

- 文件名是否写错
- 大小写是否一致
- 图片是否真的在对应目录
- 路径是相对当前 HTML 文件计算的

### 链接没有协议

外部链接建议写完整：

```html
https://example.com
```

不要写成：

```html
example.com
```
