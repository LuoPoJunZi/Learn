# 1.5 CSS 基础样式

CSS 用来控制网页的外观，比如颜色、字体、大小、间距、背景、布局。

## 三种写 CSS 的方式

### 行内样式

```html
<p style="color: red;">红色文字</p>
```

不推荐大量使用，因为不好维护。

### 内部样式

```html
<style>
  p {
    color: red;
  }
</style>
```

适合小页面。

### 外部样式

`index.html`：

```html
<link rel="stylesheet" href="css/style.css">
```

`css/style.css`：

```css
p {
  color: red;
}
```

推荐正式页面使用外部 CSS。

## 选择器

标签选择器：

```css
p {
  color: #333;
}
```

类选择器：

```html
<p class="intro">介绍文字</p>
```

```css
.intro {
  font-size: 18px;
}
```

ID 选择器：

```html
<div id="app"></div>
```

```css
#app {
  max-width: 960px;
}
```

一般样式优先用 class，不要滥用 id。

## 常用样式

```css
body {
  font-family: Arial, sans-serif;
  color: #222;
  background: #f7f7f7;
}

h1 {
  font-size: 32px;
}

p {
  line-height: 1.8;
}
```

## 颜色

```css
color: red;
color: #ff0000;
color: rgb(255, 0, 0);
```

常用十六进制颜色，比如：

- `#000000` 黑色
- `#ffffff` 白色
- `#333333` 深灰
- `#f5f5f5` 浅灰

## 背景

```css
body {
  background-color: #f5f5f5;
}

.hero {
  background-image: url("../images/banner.jpg");
  background-size: cover;
  background-position: center;
}
```

## 鼠标悬停

```css
a {
  color: blue;
}

a:hover {
  color: red;
}
```

## CSS 调试建议

浏览器按 `F12` 打开开发者工具，可以查看元素结构和 CSS 样式。

新手改样式时，建议一次只改一两个属性，保存后刷新页面看效果。
