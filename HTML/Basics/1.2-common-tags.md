# 1.2 常用 HTML 标签

HTML 标签负责表达页面结构。先掌握常见标签，就能看懂大部分静态页面。

## 标题

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
```

标题从 `h1` 到 `h6`。一篇页面通常只放一个 `h1`。

## 段落

```html
<p>这是一段文字。</p>
```

段落适合放正文说明。

## 强调

```html
<strong>重要内容</strong>
<em>强调内容</em>
```

- `strong` 表示重要
- `em` 表示强调

不要只把它们理解成“加粗”和“斜体”，它们也有语义。

## 换行和分割线

```html
第一行<br>
第二行

<hr>
```

- `br` 换行
- `hr` 分割线

正文中不要滥用 `br` 做布局，布局应该交给 CSS。

## 容器 div 和 span

```html
<div>
  <h2>卡片标题</h2>
  <p>卡片内容</p>
</div>

<p>我喜欢 <span>HTML</span>。</p>
```

- `div` 是块级容器，常用于分组布局
- `span` 是行内容器，常用于包住一小段文字

## 语义化结构标签

```html
<header>页面头部</header>
<nav>导航</nav>
<main>主要内容</main>
<section>一个章节</section>
<article>一篇文章</article>
<aside>侧边内容</aside>
<footer>页面底部</footer>
```

语义化标签能让页面结构更清楚，也更利于搜索引擎和无障碍阅读。

## 注释

```html
<!-- 这里是注释，不会显示在页面上 -->
```

注释适合解释复杂结构，但不要写太多废话。

## 一个小例子

```html
<header>
  <h1>我的学习主页</h1>
  <p>记录每天学到的新知识。</p>
</header>

<main>
  <section>
    <h2>今天学习 HTML</h2>
    <p>我学会了标题、段落和语义化标签。</p>
  </section>
</main>

<footer>
  <p>© 2026 我的学习记录</p>
</footer>
```
