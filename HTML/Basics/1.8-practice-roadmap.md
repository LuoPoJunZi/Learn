# 1.8 静态页面练习路线

学 HTML、CSS、JavaScript 最好的方式，是从小页面开始做。

## 阶段一：纯 HTML 页面

目标：熟悉常用标签。

练习：

1. 做一个个人介绍页面。
2. 做一个学习笔记目录页。
3. 做一个书单页面。

页面中至少包含：

- 标题
- 段落
- 链接
- 图片
- 列表

## 阶段二：加入 CSS

目标：让页面更清楚、更好看。

练习：

1. 给页面设置字体、颜色和背景。
2. 做一个居中的内容容器。
3. 做三个卡片横向排列。
4. 做一个简单导航栏。

建议使用：

- `max-width`
- `margin: 0 auto`
- `padding`
- `display: flex`
- `display: grid`

## 阶段三：响应式页面

目标：手机和电脑都能看。

练习：

1. 桌面端三列卡片。
2. 手机端单列卡片。
3. 图片宽度不超出屏幕。

示例：

```css
img {
  max-width: 100%;
  height: auto;
}

@media (max-width: 600px) {
  .grid {
    grid-template-columns: 1fr;
  }
}
```

## 阶段四：加入 JavaScript

目标：让页面能交互。

练习：

1. 点击按钮切换文字。
2. 点击按钮切换暗色模式。
3. 做一个计数器。
4. 做一个简单 Todo 列表。

计数器示例：

```html
<button id="minus">-</button>
<span id="count">0</span>
<button id="plus">+</button>

<script>
  let count = 0;
  const countEl = document.querySelector("#count");

  document.querySelector("#plus").addEventListener("click", function () {
    count += 1;
    countEl.textContent = count;
  });

  document.querySelector("#minus").addEventListener("click", function () {
    count -= 1;
    countEl.textContent = count;
  });
</script>
```

## 学习习惯

- 每次只改一点，保存后刷新浏览器。
- 打开开发者工具看 Console 报错。
- 文件名尽量用英文和小写。
- 图片放在 `images` 目录。
- CSS 放在 `css` 目录。
- JavaScript 放在 `js` 目录。
