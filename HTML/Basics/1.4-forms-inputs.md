# 1.4 表单与用户输入

表单用于收集用户输入，比如登录、搜索、留言、报名。

## 基本表单

```html
<form>
  <label for="name">姓名</label>
  <input id="name" name="name" type="text">

  <button type="submit">提交</button>
</form>
```

## label 的作用

`label` 用来说明输入框含义。

```html
<label for="email">邮箱</label>
<input id="email" type="email">
```

`for` 要对应输入框的 `id`。点击文字时，浏览器会自动聚焦到输入框。

## 常见 input 类型

```html
<input type="text">
<input type="password">
<input type="email">
<input type="number">
<input type="date">
<input type="checkbox">
<input type="radio">
<input type="file">
```

## 文本区域

```html
<label for="message">留言</label>
<textarea id="message" name="message"></textarea>
```

`textarea` 适合多行文本。

## 下拉选择

```html
<label for="city">城市</label>
<select id="city" name="city">
  <option value="shanghai">上海</option>
  <option value="beijing">北京</option>
  <option value="shenzhen">深圳</option>
</select>
```

## 表单校验

```html
<input type="email" required>
<input type="text" minlength="2" maxlength="20">
<input type="number" min="0" max="100">
```

浏览器会做基础校验，但真正重要的数据校验还需要后端处理。

## 按钮

```html
<button type="submit">提交</button>
<button type="reset">重置</button>
<button type="button">普通按钮</button>
```

如果按钮只是触发 JavaScript，使用 `type="button"`。

## 一个完整例子

```html
<form>
  <div>
    <label for="username">用户名</label>
    <input id="username" name="username" type="text" required>
  </div>

  <div>
    <label for="email">邮箱</label>
    <input id="email" name="email" type="email" required>
  </div>

  <div>
    <label for="message">留言</label>
    <textarea id="message" name="message"></textarea>
  </div>

  <button type="submit">提交</button>
</form>
```

## 新手提醒

纯静态 HTML 表单不会自动把数据保存到服务器。如果没有后端或第三方服务，点击提交后不会真正保存数据。
