# 1.1 网页是什么与基本结构

网页本质上是一份文本文件，浏览器读取 HTML、CSS 和 JavaScript 后，把它渲染成你看到的页面。

## 第一个 HTML 文件

新建文件 `index.html`：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>我的第一个网页</title>
</head>
<body>
  <h1>你好，HTML</h1>
  <p>这是我的第一个网页。</p>
</body>
</html>
```

双击 `index.html`，用浏览器打开，就能看到页面。

## 基本结构解释

```html
<!DOCTYPE html>
```

告诉浏览器：这是一个 HTML5 页面。

```html
<html lang="zh-CN">
```

网页根元素，`lang="zh-CN"` 表示页面主要语言是简体中文。

```html
<head>
```

放页面配置，比如编码、标题、移动端适配、CSS 引入等。`head` 里的内容通常不会直接显示在页面正文中。

```html
<body>
```

放页面正文。用户看到的大部分内容都在 `body` 里。

## 标签是什么

HTML 使用标签描述内容。

```html
<p>这是一段文字</p>
```

多数标签成对出现：

- 开始标签：`<p>`
- 结束标签：`</p>`
- 内容：`这是一段文字`

有些标签是自闭合或没有结束标签，例如：

```html
<img src="image.jpg" alt="图片说明">
<br>
<hr>
```

## 属性是什么

属性写在开始标签里，用来补充信息。

```html
<a href="https://example.com">打开网站</a>
```

这里：

- `a` 是链接标签
- `href` 是属性
- `https://example.com` 是属性值

## 推荐文件结构

简单网页可以这样组织：

```text
my-page/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── images/
    └── banner.jpg
```

刚开始可以把 HTML、CSS、JS 都写在一个文件里；熟悉后再拆分。
