# 1.6 盒模型与页面布局

网页上的每个元素都可以看成一个盒子。理解盒模型后，布局会清楚很多。

## 盒模型

一个盒子包括：

- content：内容
- padding：内边距
- border：边框
- margin：外边距

示例：

```css
.card {
  width: 300px;
  padding: 20px;
  border: 1px solid #ddd;
  margin: 16px;
}
```

## box-sizing

推荐设置：

```css
* {
  box-sizing: border-box;
}
```

这样 `width` 会包含 padding 和 border，更容易控制布局。

## 块级和行内元素

块级元素通常独占一行：

- `div`
- `p`
- `h1`
- `section`

行内元素不会独占一行：

- `span`
- `a`
- `strong`

## Flex 布局

Flex 适合一维布局，比如横向排列卡片、按钮、导航。

```css
.nav {
  display: flex;
  gap: 16px;
  align-items: center;
  justify-content: space-between;
}
```

常用属性：

- `display: flex`
- `gap`
- `align-items`
- `justify-content`
- `flex-wrap`

## Grid 布局

Grid 适合二维布局，比如卡片网格。

```css
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
```

响应式写法：

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
  gap: 16px;
}
```

## 居中

水平居中块：

```css
.container {
  max-width: 960px;
  margin: 0 auto;
}
```

Flex 居中：

```css
.center {
  display: flex;
  align-items: center;
  justify-content: center;
}
```

## 响应式基础

使用媒体查询：

```css
@media (max-width: 600px) {
  .grid {
    grid-template-columns: 1fr;
  }
}
```

这表示屏幕宽度小于 600px 时，改成单列布局。

## 新手布局建议

- 页面外层用 `.container` 控制最大宽度。
- 横向排列用 Flex。
- 卡片列表用 Grid。
- 不要用大量 `<br>` 和空格做布局。
- 优先让页面在手机上也能正常阅读。
