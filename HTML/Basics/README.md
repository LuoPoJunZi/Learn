这是一套写给网页开发新手的基础教程。它会从“一个网页由什么组成”开始，逐步讲到 HTML 标签、CSS 样式、页面布局、表单和 JavaScript 基础交互。

## 适合谁

- 从来没写过网页的新手
- 想看懂 `Birthday`、`Love` 示例页面的人
- 想自己改网页文字、颜色、图片和动画的人
- 想学习 HTML、CSS、JavaScript 三件套的人

## 一个网页的三层

| 层 | 技术 | 作用 |
| :--- | :--- | :--- |
| 结构 | HTML | 页面有什么内容 |
| 样式 | CSS | 页面长什么样 |
| 行为 | JavaScript | 页面怎么交互 |

你可以把它理解成：

- HTML 是骨架
- CSS 是外观
- JavaScript 是动作
