# HTML 示例页面

这个目录保存一些 HTML 网页示例，适合学习网页结构、样式、脚本交互和简单页面改造。

## 目录说明

- [Basics](Basics/README.md)：HTML、CSS、JavaScript 基础教程
- [Examples](Examples/README.md)：作品集、产品落地页、餐厅展示页、数据面板、交互组件和价格页案例
- [Birthday](Birthday/README.md)：生日快乐网页示例
- [Love](Love/README.md)：表白网页示例
- [优秀外部项目导读](EXTERNAL_REPOSITORIES.md)：适合继续练习的 HTML/CSS/JS 开源项目和模板资源

## 如何查看

这些页面都是静态 HTML 文件，通常可以直接双击 `.html` 文件在浏览器中打开。

如果页面包含本地脚本或资源，建议保持目录结构不变。

新增的 [Examples](Examples/README.md) 案例都使用纯 HTML/CSS/JavaScript 编写，不需要安装 npm、框架或构建工具。

## 学习建议

阅读示例时可以按这个顺序看：

1. `html` 结构：页面有哪些元素。
2. `style` 或 CSS：页面样式如何设置。
3. `script` 或 JS：动画、点击、交互逻辑在哪里。
4. 文案和图片：哪些内容可以替换成自己的。

如果你还不熟悉网页基础，建议先看 [HTML 基础教程](Basics/README.md)，再改示例页面。

如果你已经能看懂基础标签，可以继续看 [Examples](Examples/README.md)：先改文案和颜色，再尝试调整布局，最后再研究交互脚本。
