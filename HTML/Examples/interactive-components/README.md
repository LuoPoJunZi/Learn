# 交互组件集合

这个案例集合包含几个常见 JavaScript 小组件：Tabs、FAQ 折叠、Toast 提示和进度步骤。

## 学习目标

- 使用 `querySelector` 获取元素。
- 使用 `addEventListener` 绑定事件。
- 切换 class 控制显示状态。
- 用少量 JavaScript 驱动页面交互。

## 如何打开

双击 [index.html](index.html) 即可在浏览器中查看。

## 推荐修改点

- 增加一个 Tab。
- 增加一个 FAQ 问题。
- 修改 Toast 文案。
- 增加进度步骤数量。

## 参考来源

本案例为原创轻量示例，交互方向参考常见前端练习项目，没有复制外部模板源码。
