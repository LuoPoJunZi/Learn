# HTML 案例集合

这个目录保存一组轻量、原创、适合初学者学习和改造的静态网页案例。所有案例默认使用纯 HTML、CSS、JavaScript，不依赖 npm、构建工具、框架或远程 CDN。

## 如何查看

直接双击每个案例目录里的 `index.html`，或者用浏览器打开即可。

如果你使用 VS Code，可以安装 Live Server 扩展，但不是必须。

## 推荐学习方式

1. 先打开页面看效果。
2. 再看 `index.html`，找页面结构。
3. 再看 `style.css`，理解布局和视觉样式。
4. 如果有 `script.js`，最后看交互逻辑。
5. 每次只改一小处，例如颜色、文案、卡片数量或按钮动作。

## 修改练习

- 把作品集案例改成自己的简历。
- 把产品落地页改成一个真实工具或课程。
- 把餐厅页改成奶茶店、书店或工作室。
- 给数据面板增加一行表格。
- 给交互组件增加一个 FAQ 问题。
- 给价格页增加一个“团队版”套餐。
