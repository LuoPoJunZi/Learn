# 2.1 图像处理入门

Matlab 很适合做图像读取、显示、灰度化、二值化、滤波和边缘检测。新手学习图像处理时，先把“图像其实是矩阵”这个概念理解清楚。

## 学习目标

学完后你应该能：

- 读取一张图片。
- 查看图片矩阵大小。
- 显示原图、灰度图和二值图。
- 理解像素、通道、灰度值的含义。
- 知道图像处理案例常见的输入和输出。

## 图像是什么

在 Matlab 里，图像通常可以理解成矩阵：

- 灰度图：二维矩阵，每个元素表示亮度。
- 彩色图：三维数组，通常包含 R、G、B 三个通道。

查看图像大小：

```matlab
img = imread("peppers.png");
size(img)
```

如果输出类似 `384 512 3`，表示高度 384、宽度 512、3 个颜色通道。

## 读取和显示图像

```matlab
img = imread("peppers.png");
imshow(img)
title("原始图像")
```

如果当前目录没有图片，可以先使用 Matlab 自带示例图，或者把自己的图片放到当前工作目录。

## 彩色图转灰度图

```matlab
gray = rgb2gray(img);
imshow(gray)
title("灰度图")
```

灰度图只有亮度信息，很多边缘检测、阈值分割算法会先转灰度。

## 二值化

```matlab
bw = imbinarize(gray);
imshow(bw)
title("二值图")
```

二值图只有 0 和 1，常用于前景/背景分割。

## 边缘检测

```matlab
edges = edge(gray, "Canny");
imshow(edges)
title("边缘检测")
```

边缘检测会突出亮度变化明显的位置。

## 常见问题

| 问题 | 检查方向 |
| :--- | :--- |
| 图片读不到 | 当前工作目录、文件名、扩展名。 |
| `rgb2gray` 报错 | 输入可能已经是灰度图。 |
| 图像显示很暗 | 数据类型或数值范围可能不合适。 |
| 函数不存在 | 可能缺少 Image Processing Toolbox。 |

## 推荐练习

1. 读取一张自己的图片。
2. 转成灰度图。
3. 转成二值图。
4. 做一次边缘检测。
5. 分别保存结果图。

## 下一步

图像处理案例通常会涉及工具箱函数。运行前先确认是否安装 Image Processing Toolbox。基础概念掌握后，可以再学习滤波、形态学处理、目标检测等内容。
