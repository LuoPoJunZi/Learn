# 1.1 Matlab 环境与命令窗口

Matlab 是一款面向矩阵计算、科学计算、工程仿真和数据可视化的软件。新手刚打开 Matlab 时，最重要的是先认识几个区域。

## 常见界面区域

### Command Window

命令窗口。你可以直接输入命令并立即运行。

```matlab
1 + 2
sqrt(16)
```

### Workspace

工作区。这里会显示当前创建过的变量。

```matlab
a = 10;
b = 20;
c = a + b;
```

运行后，`a`、`b`、`c` 会出现在 Workspace 中。

### Current Folder

当前文件夹。Matlab 运行脚本时，会优先从当前文件夹查找 `.m` 文件和数据文件。

如果脚本提示找不到文件，第一件事就是检查 Current Folder 是否正确。

### Editor

编辑器。用于编写 `.m` 脚本和函数。

## 第一条命令

```matlab
disp("Hello Matlab")
```

`disp` 用来显示内容。

## 常用清理命令

```matlab
clc
clear
close all
```

含义：

- `clc` 清空命令窗口显示
- `clear` 清空工作区变量
- `close all` 关闭所有图像窗口

很多脚本开头会写：

```matlab
clc;
clear;
close all;
```

这样可以减少旧变量和旧图像对新运行结果的影响。

## 分号的作用

不加分号会显示结果：

```matlab
a = 10
```

加分号不显示结果：

```matlab
a = 10;
```

实际写脚本时，通常给中间变量加分号，让输出更干净。

## 查看变量

```matlab
who
whos
```

- `who` 显示变量名
- `whos` 显示变量名、大小、类型、占用内存等信息

## 查看帮助

```matlab
help plot
doc plot
```

- `help` 在命令窗口显示简要帮助
- `doc` 打开官方文档页面

## 新手常见问题

### 脚本为什么找不到数据文件

通常是当前文件夹不对。把 Current Folder 切换到脚本所在目录，或者使用完整路径。

### 为什么变量结果没有显示

可能是代码行末尾加了分号 `;`。分号会抑制输出，但变量仍然创建了。

### 为什么之前的变量影响了现在的结果

运行脚本前可以先执行：

```matlab
clear
```

或者脚本开头写 `clear;`。
