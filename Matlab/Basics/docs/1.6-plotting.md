# 1.6 基础绘图

Matlab 很适合快速画图。新手先掌握二维曲线、散点图、柱状图和多图布局。

## plot 曲线图

```matlab
x = 0:0.1:2*pi;
y = sin(x);

plot(x, y)
```

注意这里要用：

```matlab
y = sin(x);
```

如果自己写平方，要使用点运算：

```matlab
y = x.^2;
```

## 添加标题和坐标轴

```matlab
plot(x, y)
title("Sine Wave")
xlabel("x")
ylabel("sin(x)")
grid on
```

## 多条曲线

```matlab
x = 0:0.1:2*pi;
y1 = sin(x);
y2 = cos(x);

plot(x, y1, x, y2)
legend("sin(x)", "cos(x)")
```

## 设置线条样式

```matlab
plot(x, y, "r--", "LineWidth", 2)
```

常见样式：

- `r` 红色
- `b` 蓝色
- `--` 虚线
- `o` 圆点

## 散点图

```matlab
x = rand(1, 50);
y = rand(1, 50);
scatter(x, y)
```

## 柱状图

```matlab
data = [3 7 5 9];
bar(data)
```

## 子图 subplot

```matlab
x = 0:0.1:2*pi;

subplot(2, 1, 1)
plot(x, sin(x))
title("sin")

subplot(2, 1, 2)
plot(x, cos(x))
title("cos")
```

`subplot(2, 1, 1)` 表示 2 行 1 列中的第 1 张图。

## 保存图片

```matlab
saveas(gcf, "figure.png")
```

`gcf` 表示当前图像窗口。
