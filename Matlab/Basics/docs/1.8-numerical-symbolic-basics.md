# 1.8 数值计算与符号计算入门

Matlab 既能做数值计算，也能做一部分符号计算。新手要先理解两者区别。

## 数值计算

数值计算得到具体数字。

```matlab
sqrt(2)
sin(pi / 2)
```

数值计算适合工程计算、仿真、数据处理。

## 多项式

Matlab 用系数向量表示多项式。

例如：

```text
x^2 - 3x + 2
```

表示为：

```matlab
p = [1 -3 2];
```

求根：

```matlab
roots(p)
```

计算多项式值：

```matlab
polyval(p, 2)
```

## 数值积分

定义函数：

```matlab
f = @(x) x.^2;
```

积分：

```matlab
integral(f, 0, 1)
```

## 数值求根

```matlab
f = @(x) cos(x) - x;
root = fzero(f, 0);
```

`0` 是初始点。

## 符号计算

符号计算需要 Symbolic Math Toolbox。

```matlab
syms x
diff(x^2 + 3*x + 1, x)
int(x^2, x)
solve(x^2 - 4 == 0, x)
```

含义：

- `syms x` 定义符号变量
- `diff` 求导
- `int` 积分
- `solve` 解方程

## 符号计算和数值计算对比

| 需求 | 推荐 |
| :--- | :--- |
| 得到公式结果 | 符号计算 |
| 得到具体数值 | 数值计算 |
| 工程仿真 | 数值计算 |
| 数学推导 | 符号计算 |

## 线性方程组

数值解：

```matlab
A = [1 2; 3 4];
b = [5; 11];
x = A \ b;
```

符号解：

```matlab
syms x y
solve([x + 2*y == 5, 3*x + 4*y == 11], [x, y])
```

## 小练习

1. 用 `roots` 求 `x^2 - 5x + 6 = 0` 的根
2. 用 `integral` 计算 `x^2` 在 `[0, 1]` 上的积分
3. 用 `fzero` 求 `cos(x) - x = 0`
4. 如果有符号工具箱，用 `solve` 解同一个方程
