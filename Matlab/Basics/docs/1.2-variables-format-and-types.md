# 1.2 变量、显示格式与常用类型

Matlab 中变量不需要提前声明，直接赋值即可。

## 创建变量

```matlab
a = 10;
b = 3.14;
name = "Matlab";
```

变量名建议：

- 使用英文
- 不以数字开头
- 不使用空格
- 不和内置函数重名

## 避免变量名覆盖函数名

不建议这样写：

```matlab
cos = "hello";
```

这样会让 `cos` 不再代表余弦函数，而是一个变量。

清除变量后恢复：

```matlab
clear cos
```

## 查看关键字

```matlab
iskeyword
```

这些关键字不能作为变量名，比如 `for`、`if`、`end` 等。

## 数字和常量

```matlab
pi
Inf
NaN
eps
i
j
```

含义：

- `pi` 圆周率
- `Inf` 无穷大
- `NaN` 不是一个数
- `eps` 浮点精度附近的很小数
- `i`、`j` 默认可表示虚数单位

## 显示格式 format

`format` 只影响显示方式，不改变实际数值。

```matlab
format short
pi

format long
pi

format bank
pi

format rat
pi
```

常见格式：

| 命令 | 作用 |
| :--- | :--- |
| `format short` | 默认短小数 |
| `format long` | 显示更多小数位 |
| `format shortE` | 科学计数法 |
| `format longE` | 更多位科学计数法 |
| `format bank` | 两位小数 |
| `format rat` | 分数形式 |

## 字符串和字符数组

字符串：

```matlab
s = "hello";
```

字符数组：

```matlab
c = 'hello';
```

新手可以优先使用双引号字符串。

## 类型查看

```matlab
class(a)
size(a)
length(a)
```

- `class` 查看类型
- `size` 查看数组维度
- `length` 查看最大维度长度

## 小练习

1. 创建变量 `radius = 3`
2. 计算圆面积 `area = pi * radius^2`
3. 用 `format long` 显示结果
4. 用 `whos` 查看变量信息

参考：

```matlab
clc;
clear;

radius = 3;
area = pi * radius^2;

format long
area
whos
```
