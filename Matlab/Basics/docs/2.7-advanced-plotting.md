# 2.7 高阶绘图入门

基础绘图解决“画出来”的问题，高阶绘图解决“看得懂、能比较、能表达”的问题。本节整理多子图、双坐标轴、误差条、直方图、三维图等常见画法。

## 学习目标

学完后你应该能：

- 用 `subplot` 在一张图里放多个图。
- 用 `hold on` 对比多条曲线。
- 给图添加标题、坐标轴、图例和网格。
- 画直方图、误差条、三维曲面图。
- 知道图像保存和结果解释的基本方法。

## 多条曲线对比

```matlab
x = 0:0.1:2*pi;
y1 = sin(x);
y2 = cos(x);

plot(x, y1, "LineWidth", 1.5)
hold on
plot(x, y2, "LineWidth", 1.5)
hold off

grid on
xlabel("x")
ylabel("y")
title("sin 和 cos 对比")
legend("sin(x)", "cos(x)")
```

`hold on` 表示保留当前图，再继续画下一条曲线。画完后用 `hold off` 关闭叠加状态。

## 子图 subplot

```matlab
x = 0:0.1:2*pi;

subplot(2, 1, 1)
plot(x, sin(x))
title("sin(x)")
grid on

subplot(2, 1, 2)
plot(x, cos(x))
title("cos(x)")
grid on
```

`subplot(2, 1, 1)` 表示把图窗分成 2 行 1 列，当前画第 1 个位置。

## 双坐标轴

当两组数据量级差别很大时，可以使用左右两个 y 轴：

```matlab
x = 0:0.1:10;
y1 = exp(-0.2*x) .* sin(x);
y2 = 100 * exp(-0.2*x);

yyaxis left
plot(x, y1)
ylabel("振荡信号")

yyaxis right
plot(x, y2)
ylabel("衰减包络")

xlabel("x")
title("双 y 轴示例")
grid on
```

双坐标轴容易误导读者，使用时一定要写清两个 y 轴分别代表什么。

## 直方图

直方图用于观察数据分布：

```matlab
data = randn(1, 1000);
histogram(data, 30)
title("数据分布")
xlabel("数值")
ylabel("频数")
grid on
```

`30` 表示分箱数量。分箱太少会丢细节，太多会显得很碎。

## 误差条

误差条用于展示测量误差或波动范围：

```matlab
x = 1:5;
y = [2.1 2.9 3.7 4.1 5.2];
err = [0.2 0.3 0.25 0.4 0.35];

errorbar(x, y, err, "o-")
grid on
xlabel("实验编号")
ylabel("测量值")
title("带误差条的结果")
```

如果图中有误差条，正文或图注中要说明误差代表什么。

## 三维曲面图

```matlab
[X, Y] = meshgrid(-2:0.1:2, -2:0.1:2);
Z = X.^2 + Y.^2;

surf(X, Y, Z)
shading interp
colorbar
xlabel("x")
ylabel("y")
zlabel("z")
title("三维曲面")
```

`meshgrid` 用于生成二维网格，`surf` 用于画曲面。

## 保存图像

```matlab
saveas(gcf, "figure-result.png")
```

保存前检查：

- 标题是否清楚。
- 坐标轴是否有标签。
- 图例是否能区分曲线。
- 文件名是否表达内容。

## 常见问题

| 问题 | 建议 |
| :--- | :--- |
| 多条线覆盖不清 | 加图例、改线型或颜色。 |
| 子图太挤 | 减少子图数量，或拆成多张图。 |
| 双坐标轴误导 | 明确标注左右 y 轴含义。 |
| 三维图看不清 | 调整视角，或改用等高线图。 |
| 保存图没有标题 | 保存前确认当前图窗就是目标图。 |
