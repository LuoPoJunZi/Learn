# 1.5 脚本与函数

Matlab 代码通常保存在 `.m` 文件里。`.m` 文件可以是脚本，也可以是函数。

## 脚本是什么

脚本是一组按顺序执行的命令。

新建 `hello.m`：

```matlab
clc;
clear;

name = "Matlab";
disp("Hello " + name)
```

在命令窗口输入：

```matlab
hello
```

Matlab 会运行 `hello.m`。

## 脚本的特点

- 使用当前工作区变量
- 运行后创建的变量会留在工作区
- 适合写实验流程、绘图、数据处理步骤

## 函数是什么

函数有输入和输出，更适合复用。

新建 `addTwo.m`：

```matlab
function result = addTwo(a, b)
    result = a + b;
end
```

调用：

```matlab
x = addTwo(3, 5)
```

## 函数文件名要匹配函数名

如果函数叫：

```matlab
function result = addTwo(a, b)
```

文件名应该是：

```text
addTwo.m
```

## 多输出函数

```matlab
function [sumValue, productValue] = calcTwo(a, b)
    sumValue = a + b;
    productValue = a * b;
end
```

调用：

```matlab
[s, p] = calcTwo(3, 5)
```

## 匿名函数

```matlab
f = @(x) x.^2 + 1;
y = f(3);
```

匿名函数适合定义简单函数，常用于绘图、数值积分和优化。

## if 语句

```matlab
score = 85;

if score >= 90
    disp("优秀")
elseif score >= 60
    disp("及格")
else
    disp("不及格")
end
```

## for 循环

```matlab
total = 0;

for i = 1:100
    total = total + i;
end

disp(total)
```

## while 循环

```matlab
n = 1;

while n < 5
    disp(n)
    n = n + 1;
end
```

## 新手建议

- 实验流程用脚本。
- 重复使用的计算写成函数。
- 函数文件名和函数名保持一致。
- 每个函数尽量只做一件清楚的事情。
