# 1.3 向量、矩阵与索引

Matlab 的名字来自 Matrix Laboratory。矩阵和数组是 Matlab 的核心。

## 行向量

```matlab
a = [1 2 3 4]
```

也可以用逗号分隔：

```matlab
a = [1, 2, 3, 4]
```

## 列向量

```matlab
b = [1; 2; 3; 4]
```

分号表示换行。

## 矩阵

```matlab
A = [1 2 3; 4 5 6; 7 8 9]
```

这是一个 3 行 3 列矩阵。

## 索引从 1 开始

```matlab
A(1, 1)
A(1, 2)
A(3, 3)
```

Matlab 的索引从 1 开始，不是从 0 开始。

## 取整行和整列

```matlab
A(1, :)   % 第 1 行所有列
A(:, 2)   % 第 2 列所有行
```

冒号 `:` 表示“全部”。

## 修改元素

```matlab
A(1, 1) = 100;
A(2, :) = [10 20 30];
```

## 删除行或列

删除第 2 行：

```matlab
A(2, :) = [];
```

删除第 3 列：

```matlab
A(:, 3) = [];
```

## 冒号生成序列

```matlab
1:10
1:2:9
10:-1:1
```

含义：

- `1:10` 从 1 到 10
- `1:2:9` 从 1 到 9，步长为 2
- `10:-1:1` 从 10 到 1，步长为 -1

## linspace

```matlab
linspace(0, 1, 5)
```

生成从 0 到 1 的 5 个等间距点。

## 常用特殊矩阵

```matlab
zeros(3, 4)
ones(2, 3)
eye(3)
rand(2, 2)
diag([1 2 3])
```

## 矩阵拼接

横向拼接：

```matlab
A = [1 2; 3 4];
B = [5 6; 7 8];
C = [A B];
```

纵向拼接：

```matlab
C = [A; B];
```
