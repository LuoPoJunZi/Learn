# 2.6 字符串、结构体与单元格数组

Matlab 除了数值矩阵，还经常会用到字符串、结构体和单元格数组。它们适合保存“不是纯数字矩阵”的数据，例如姓名、参数、实验记录、不同长度的数据列。

## 学习目标

学完后你应该能：

- 区分字符数组和字符串。
- 用结构体保存一组带名字的字段。
- 用单元格数组保存不同类型或不同大小的数据。
- 知道什么时候用矩阵、结构体、单元格数组。

## 字符数组和字符串

早期 Matlab 常用单引号表示字符数组：

```matlab
name1 = 'Matlab';
```

较新版本推荐使用双引号表示字符串：

```matlab
name2 = "Matlab";
```

新手可以优先使用双引号字符串，因为它更接近“一个文本值”的直觉。

## 字符串拼接

```matlab
first = "Hello";
second = "Matlab";
message = first + ", " + second;
```

如果使用字符数组，拼接常用方括号：

```matlab
message = ['Hello', ', ', 'Matlab'];
```

## 结构体

结构体适合保存一组相关信息，每个信息都有字段名。

```matlab
student.name = "Alice";
student.score = 92;
student.passed = true;
```

读取字段：

```matlab
student.name
student.score
```

结构体很适合保存实验配置：

```matlab
config.learningRate = 0.01;
config.epochs = 100;
config.model = "BP";
```

这样比散落很多变量更清楚。

## 结构体数组

如果有多个学生，可以使用结构体数组：

```matlab
students(1).name = "Alice";
students(1).score = 92;

students(2).name = "Bob";
students(2).score = 85;
```

访问第二个学生：

```matlab
students(2).name
```

## 单元格数组

普通矩阵要求元素类型和大小比较统一。单元格数组可以保存不同类型的数据。

```matlab
C = {42, "Matlab", [1 2 3]};
```

访问单元格有两种方式：

```matlab
C(1)   % 得到一个 cell
C{1}   % 得到 cell 里面的内容
```

这是新手最容易混淆的地方：

- `()` 取出来还是盒子。
- `{}` 取出来是盒子里的东西。

## 什么时候用哪种数据结构

| 场景 | 推荐 |
| :--- | :--- |
| 纯数字表格或矩阵运算 | 数值矩阵 |
| 一组有名字的参数 | 结构体 |
| 多个实验样本，每个样本字段相同 | 结构体数组 |
| 不同类型、不同长度的数据放一起 | 单元格数组 |
| 带列名的表格数据 | table |

## 常见错误

### 把 `()` 和 `{}` 混用

```matlab
C = {1, 2, 3};
a = C(1);   % a 是 cell
b = C{1};   % b 是数字 1
```

如果后续要参与数值计算，通常需要用 `{}` 取出内容。

### 字符数组长度不同导致拼接失败

```matlab
["cat"; "Matlab"]
```

字符串数组可以处理不同长度文本；字符数组按矩阵存储时更容易遇到长度限制。

### 结构体字段名写错

```matlab
student.score = 90;
student.scroe
```

字段名拼错会导致找不到字段。建议字段名保持简单、统一。

## 小例子：保存实验结果

```matlab
result.model = "linear";
result.parameters = [2, 1];
result.mse = 0.03;
result.notes = {"first run", "no normalization"};
```

这个结构体比单独保存 `model`、`parameters`、`mse`、`notes` 更容易传递和保存。

## 推荐练习

1. 创建一个 `student` 结构体，包含姓名、分数、是否通过。
2. 创建一个 `config` 结构体，保存学习率、训练轮数、模型名称。
3. 创建一个 cell，保存一个数字、一个字符串、一个向量。
4. 分别用 `()` 和 `{}` 访问 cell，观察结果区别。
