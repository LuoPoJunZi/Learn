# 2.4 插值方法入门

插值用于根据已知数据点估计中间位置的数值。它常见于实验数据补点、曲线平滑、图像缩放和数值计算。

## 学习目标

学完后你应该能：

- 理解插值和拟合的区别。
- 使用 `interp1` 做一维插值。
- 对比线性插值和样条插值。
- 知道插值不等于预测未来。

## 插值和拟合的区别

插值通常要求曲线经过已知数据点。

拟合更关注整体趋势，不一定经过每个点。

简单记忆：

```text
插值：补中间点
拟合：找整体规律
```

## 一维线性插值

```matlab
x = [0 1 2 3 4];
y = [0 1 4 9 16];

xq = 0:0.1:4;
yq = interp1(x, y, xq, "linear");

plot(x, y, "o", xq, yq, "-")
legend("原始点", "线性插值")
grid on
```

`xq` 是你想估计的新位置。

## 样条插值

```matlab
yqSpline = interp1(x, y, xq, "spline");

plot(x, y, "o", xq, yq, "-", xq, yqSpline, "--")
legend("原始点", "线性插值", "样条插值")
grid on
```

样条插值通常更平滑，但不代表一定更真实。数据噪声较大时，过度平滑可能误导判断。

## 外推要谨慎

如果查询点超出原始数据范围，就变成外推。外推风险更高，因为你在已知数据之外猜测。

```matlab
interp1(x, y, 5, "linear", "extrap")
```

新手阶段不建议轻易依赖外推结果。

## 常见问题

| 问题 | 检查方向 |
| :--- | :--- |
| 插值结果是 `NaN` | 查询点可能超出范围。 |
| 曲线不平滑 | 线性插值本来就是分段直线。 |
| 样条曲线过冲 | 数据点少或变化剧烈。 |
| `x` 报错 | `x` 通常需要单调且长度与 `y` 一致。 |

## 下一步

掌握一维插值后，可以继续学习二维插值、曲面插值和图像缩放。做真实数据处理时，先画原始点，再决定是否插值。
