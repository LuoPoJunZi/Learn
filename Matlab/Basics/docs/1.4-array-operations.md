# 1.4 矩阵运算与点运算

Matlab 里最容易让新手困惑的是：`*` 和 `.*` 有什么区别？`^` 和 `.^` 有什么区别？

## 矩阵运算

```matlab
A = [1 2; 3 4];
B = [5 6; 7 8];
```

矩阵加法：

```matlab
A + B
```

矩阵乘法：

```matlab
A * B
```

矩阵乘法要求维度匹配。

## 点乘

逐元素相乘：

```matlab
A .* B
```

含义是：

```text
对应位置的元素分别相乘
```

## 点除

```matlab
A ./ B
```

逐元素相除。

## 点幂

```matlab
A .^ 2
```

矩阵中每个元素分别平方。

## 矩阵幂

```matlab
A ^ 2
```

表示矩阵乘法：

```matlab
A * A
```

只有方阵才适合做矩阵幂。

## 转置

```matlab
A'
```

如果包含复数，`'` 是共轭转置。

普通转置：

```matlab
A.'
```

## 左除和右除

解线性方程组时常用左除：

```matlab
A = [1 2; 3 4];
b = [5; 11];
x = A \ b;
```

这比 `inv(A) * b` 更推荐。

## 常见错误

### 维度不匹配

```matlab
A * B
```

如果 A 的列数不等于 B 的行数，就会报错。

### 忘记点运算

想画函数时常见错误：

```matlab
x = 0:0.1:10;
y = x^2;      % 错误
```

正确：

```matlab
y = x.^2;
```

因为 `x` 是一个向量，需要对每个元素平方。

## 小练习

```matlab
A = [1 2; 3 4];
B = [2 2; 2 2];
```

分别计算：

1. `A * B`
2. `A .* B`
3. `A ^ 2`
4. `A .^ 2`

观察结果有什么不同。
