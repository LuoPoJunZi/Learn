# 2.3 统计与回归入门

统计和回归常用于理解数据关系，例如变量之间是否相关、能否用一条线或曲线预测结果。Matlab 提供了很多内置函数，新手先从描述统计和线性回归开始。

## 学习目标

学完后你应该能：

- 计算均值、标准差、最大值和最小值。
- 画散点图观察变量关系。
- 做简单线性拟合。
- 理解残差和拟合效果。

## 描述统计

```matlab
x = [1 2 3 4 5];
mean(x)
std(x)
min(x)
max(x)
```

这些指标帮助你快速了解数据的大致范围和波动程度。

## 散点图

```matlab
x = [1 2 3 4 5];
y = [2.1 4.0 6.2 7.9 10.1];

scatter(x, y)
grid on
xlabel("x")
ylabel("y")
title("散点图")
```

做回归前，先画图。图能告诉你关系是否接近线性、是否有异常点。

## 线性拟合

```matlab
p = polyfit(x, y, 1);
yfit = polyval(p, x);

scatter(x, y)
hold on
plot(x, yfit, "LineWidth", 1.5)
hold off
legend("数据", "线性拟合")
grid on
```

`polyfit(x, y, 1)` 表示用一次多项式拟合，也就是直线拟合。

## 残差

残差是预测值和真实值的差：

```matlab
residual = y - yfit;
```

残差越小，说明拟合越接近数据。但只看训练数据上的残差不够，真实建模还需要验证集或测试集。

## 常见误区

| 误区 | 正确理解 |
| :--- | :--- |
| 相关就是因果 | 相关只说明一起变化，不等于一个导致另一个。 |
| 拟合线好看就一定可靠 | 还要看数据量、异常点和验证结果。 |
| 阶数越高越好 | 阶数太高可能过拟合。 |
| 不画图直接建模 | 先画散点图能发现很多问题。 |
