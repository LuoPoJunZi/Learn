# 1.7 数据读写

Matlab 经常需要读取数据文件、保存计算结果。新手先掌握 `.mat`、`.txt`、`.csv`、`.xlsx`。

## 保存 mat 文件

```matlab
a = 10;
b = [1 2 3];

save("data.mat", "a", "b")
```

读取：

```matlab
load("data.mat")
```

`.mat` 是 Matlab 自己的数据格式，适合保存变量。

## 读取文本文件

```matlab
text = fileread("notes.txt");
disp(text)
```

## 写入文本文件

```matlab
fid = fopen("output.txt", "w");
fprintf(fid, "Hello Matlab\n");
fclose(fid);
```

注意：打开文件后要 `fclose`。

## 读取 CSV

```matlab
data = readmatrix("data.csv");
```

如果有表头：

```matlab
T = readtable("data.csv");
```

## 读取 Excel

```matlab
data = readmatrix("data.xlsx");
T = readtable("data.xlsx");
```

`readmatrix` 适合纯数字数据，`readtable` 适合带表头的数据。

## 写入 Excel 或 CSV

```matlab
writematrix(data, "output.csv");
writetable(T, "output.xlsx");
```

## 当前路径

查看当前目录：

```matlab
pwd
```

切换目录：

```matlab
cd("D:\data")
```

推荐做法：把脚本和数据放在同一个项目文件夹里，再把 Current Folder 切换到该文件夹。

## 判断文件是否存在

```matlab
if isfile("data.xlsx")
    data = readmatrix("data.xlsx");
else
    disp("文件不存在")
end
```

## 常见问题

### 找不到文件

检查：

- 文件名是否写错
- 扩展名是否正确
- Current Folder 是否正确
- 路径里是否有中文或特殊字符

### Excel 读取结果不对

如果表格里有文字表头，优先试：

```matlab
readtable("data.xlsx")
```

如果是纯数字矩阵，使用：

```matlab
readmatrix("data.xlsx")
```
