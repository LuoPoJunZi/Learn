# Matlab 常见问题

这份 FAQ 帮助新手理解 Matlab 运行案例时最常见的问题：路径、矩阵维度、工具箱、数据文件和绘图。

## 为什么提示找不到文件？

Matlab 读取相对路径时，会从当前工作目录查找文件。当前目录不对，就会找不到数据。

先检查：

```matlab
pwd
dir
```

建议把当前目录切换到案例所在文件夹，或者在代码里使用完整路径。

## `*` 和 `.*` 有什么区别？

`*` 是矩阵乘法，要求矩阵维度满足乘法规则。

`.*` 是逐元素乘法，要求数组大小兼容。

```matlab
A * B
A .* B
```

很多维度报错都来自把矩阵运算和数组运算混在一起。

## 为什么提示矩阵维度不一致？

先看变量大小：

```matlab
size(A)
size(B)
```

如果是矩阵乘法，检查 `A` 的列数是否等于 `B` 的行数。如果是逐元素运算，检查两个数组大小是否兼容。

## 脚本和函数有什么区别？

脚本直接使用当前工作区变量，函数有自己的输入和输出。

大型案例建议把核心算法写成函数，这样更容易复用和调试。

## 为什么缺少工具箱？

有些案例需要额外工具箱，例如：

- Deep Learning Toolbox
- Statistics and Machine Learning Toolbox
- Optimization Toolbox
- Parallel Computing Toolbox

可以在 Matlab 中查看已安装工具箱，或根据报错中的函数名判断缺少哪个工具箱。

## 为什么同一个算法每次结果不一样？

优化算法和神经网络常包含随机初始化。每次运行结果略有不同是正常的。

如果想复现实验，可以设置随机种子：

```matlab
rng(1)
```

## 图画出来了，但看不懂怎么办？

至少补齐：

```matlab
title("结果")
xlabel("x")
ylabel("y")
legend("方法 A")
grid on
```

图不是为了好看而已，它要帮助你解释结果。

## 修改算法案例时应该先改哪里？

建议顺序：

1. 先跑通默认示例。
2. 找到输入数据。
3. 找到目标函数或模型入口。
4. 找到关键参数。
5. 一次只改一个参数。

如果一口气改很多地方，出错后很难定位。
