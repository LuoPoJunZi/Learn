# Matlab 示例运行指南

这篇文档写给第一次运行本仓库 Matlab 示例的新手。很多 Matlab 报错并不是算法本身有问题，而是工作目录、数据文件、工具箱或路径设置不对。

## 先理解几个概念

### 当前文件夹

Matlab 左侧或上方显示的 Current Folder，就是当前工作目录。脚本中使用相对路径读取 `.mat`、`.xlsx`、图片或函数文件时，通常都依赖这个目录。

建议：运行某个案例前，先把 Current Folder 切换到该案例所在目录。

### 脚本和函数

| 类型 | 特点 |
| :--- | :--- |
| 脚本 | 直接运行，变量会出现在工作区 |
| 函数 | 需要传入参数，通常被其他脚本调用 |

如果你不知道哪个文件是入口，先看当前目录下的 `README.md` 或 `*-USE.md`。

### 数据文件

常见数据文件包括：

- `.mat`：Matlab 原生数据文件
- `.xlsx`：Excel 数据文件
- `.tif` / `.jpg`：图像数据或结果图

不要随意移动这些文件。很多示例默认它们和 `.m` 文件在同一目录或固定子目录中。

## 推荐运行流程

1. 打开 Matlab。
2. 在 Matlab 中打开本仓库的 `Matlab` 目录。
3. 进入你想运行的具体案例目录。
4. 阅读该目录下的 `README.md` 或 `*-USE.md`。
5. 找到主脚本，例如 `main.m`、`run.m` 或说明文档指定的入口。
6. 点击 Run 或在命令窗口输入脚本名运行。
7. 观察命令窗口输出、图像窗口和生成的结果文件。

不要一开始就批量运行所有文件。先跑通一个最小示例，再逐步改参数。

## 多目标优化示例怎么跑

多目标优化目录在：

- [Multi-Objective Optimization](<Multi-Objective Optimization/README.md>)

建议顺序：

1. 先看 [算法索引](<Multi-Objective Optimization/ALGORITHM_INDEX.md>)，确认算法名称和用途。
2. 进入具体算法目录，例如 `NSGA-II`、`MOEA-D`、`MOGWO`。
3. 阅读该目录下的 `README.md` 或 `*-USE.md`。
4. 找到主脚本和测试函数。
5. 先使用默认参数运行一次。
6. 再修改种群数量、迭代次数、目标函数等参数。

常见输出包括：

- Pareto 前沿图
- 目标函数值
- 收敛曲线
- 结果矩阵或结果文件

如果算法结果每次略有不同，通常是因为包含随机初始化。需要复现实验时，可以在入口脚本中设置随机种子。

```matlab
rng(1)
```

- 是否安装了 Statistics and Machine Learning Toolbox。
- Excel 文件是否能被 Matlab 正常读取。

## 常见报错和排查

### Undefined function or variable

含义：Matlab 找不到函数、变量或文件。

排查：

- 当前文件夹是否在案例目录。
- 相关 `.m` 文件是否在当前目录或 Matlab 路径中。
- 文件名和函数名是否一致。
- 是否漏运行了前置脚本。

### Unable to read file

含义：数据文件路径不对，或文件不存在。

排查：

- `.mat`、`.xlsx`、图片文件是否还在原位置。
- 当前工作目录是否正确。
- 文件名大小写是否一致。
- 路径中是否包含特殊字符。

### Unrecognized function or variable

含义：变量还没有创建，或者函数不可用。

排查：

- 是否按说明先运行了数据加载脚本。
- 是否跳过了初始化代码。
- 是否缺少工具箱。

### No appropriate method, property, or field

含义：对象类型和代码预期不一致。

排查：

- 输入数据结构是否被改过。
- Matlab 版本是否过旧。
- 是否把表格、矩阵、结构体混用了。

### Invalid MEX-file

含义：`.mexw64` 文件无法在当前环境运行。

排查：

- `.mexw64` 通常只适用于 Windows 64 位 Matlab。
- Matlab 版本差异可能导致 MEX 文件不可用。
- macOS 或 Linux 通常不能直接运行 `.mexw64`。

## 修改参数前的建议

先记录默认参数和默认结果，再修改：

```matlab
% 示例：记录随机种子，方便复现实验
rng(1)
```

常见可修改参数：

- 种群数量
- 最大迭代次数
- 学习率
- 隐藏层神经元数量
- 训练集比例
- 目标函数
- 数据文件路径

建议一次只改一个参数。否则结果变差时，很难判断是哪一个参数导致的。

## 保存实验记录

建议为每次实验记录：

- 日期
- 算法或模型
- 数据集
- 关键参数
- 运行结果
- 图像或输出文件
- 是否成功复现

可以在案例目录中新建自己的实验笔记，但不要覆盖原始说明文档。
