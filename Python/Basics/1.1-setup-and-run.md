# 1.1 安装、运行与编辑器

学习 Python 的第一步，是确认你的电脑能运行 Python 程序。

## Python 是什么

Python 是一门通用编程语言，常用于：

- 自动化脚本
- 数据处理
- 爬虫
- Web 后端
- 人工智能和机器学习
- 文件批处理
- 日常小工具

Python 的语法相对接近英文，适合入门。

## 安装 Python

打开官网：

```text
https://www.python.org
```

下载 Python 3。建议使用 Python 3.10 或更新版本。

Windows 安装时注意勾选：

```text
Add Python to PATH
```

这个选项很重要。勾选后，你才能在终端里直接使用 `python` 或 `pip`。

## 检查是否安装成功

打开终端，输入：

```bash
python --version
```

如果输出类似：

```text
Python 3.12.0
```

说明安装成功。

有些系统需要使用：

```bash
python3 --version
```

## 第一个 Python 程序

新建文件 `hello.py`：

```python
print("Hello, Python!")
```

在终端进入文件所在目录，运行：

```bash
python hello.py
```

如果看到：

```text
Hello, Python!
```

说明你已经成功运行了第一个 Python 程序。

## 推荐编辑器

推荐使用 VS Code。

建议安装扩展：

- Python
- Pylance

VS Code 可以帮助你：

- 高亮 Python 代码
- 自动补全
- 显示语法错误
- 在终端中运行脚本

## 交互模式

直接输入：

```bash
python
```

会进入 Python 交互模式：

```text
>>>
```

你可以直接输入表达式：

```python
1 + 2
print("hi")
```

退出交互模式：

```python
exit()
```

## 脚本模式和交互模式的区别

| 模式 | 适合做什么 |
| :--- | :--- |
| 交互模式 | 快速试一行代码、测试表达式 |
| 脚本模式 | 写完整程序、保存代码、反复运行 |

新手练习时可以先用交互模式试小片段，再把有用代码写进 `.py` 文件。
