# 1.4 条件判断与循环

程序不只是从上到下一行行执行。很多时候需要根据条件做不同事情，或者重复执行某段逻辑。

## if 条件判断

```python
age = 18

if age >= 18:
    print("成年人")
```

Python 使用缩进表示代码块。通常使用 4 个空格。

## if else

```python
score = 80

if score >= 60:
    print("及格")
else:
    print("不及格")
```

## if elif else

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

## 比较运算符

```python
==  # 等于
!=  # 不等于
>   # 大于
<   # 小于
>=  # 大于等于
<=  # 小于等于
```

注意：判断相等用 `==`，赋值用 `=`。

```python
name = "Alice"      # 赋值
name == "Alice"     # 判断是否相等
```

## 逻辑运算符

```python
and  # 并且
or   # 或者
not  # 取反
```

示例：

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("可以入场")
```

## for 循环

遍历列表：

```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

遍历数字范围：

```python
for i in range(5):
    print(i)
```

输出：

```text
0
1
2
3
4
```

`range(1, 6)` 表示从 1 到 5：

```python
for i in range(1, 6):
    print(i)
```

## while 循环

当条件成立时不断执行：

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

如果忘记改变条件，可能造成死循环。

## break 和 continue

`break` 结束循环：

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

`continue` 跳过本次循环：

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

## 小练习

统计 1 到 100 的偶数和：

```python
total = 0

for i in range(1, 101):
    if i % 2 == 0:
        total += i

print(total)
```
