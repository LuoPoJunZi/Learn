# 1.3 常用数据类型

数据类型决定了一个值能做什么操作。Python 常用数据类型包括数字、字符串、布尔值、列表、元组、字典和集合。

## 数字

整数：

```python
age = 18
count = 100
```

小数：

```python
price = 19.9
height = 1.75
```

常见运算：

```python
print(10 + 3)
print(10 - 3)
print(10 * 3)
print(10 / 3)
print(10 // 3)  # 整除
print(10 % 3)   # 取余
print(2 ** 3)   # 幂运算
```

## 字符串

字符串用来保存文本。

```python
name = "Alice"
message = 'Hello'
```

常见操作：

```python
text = "Python"

print(text[0])       # P
print(text[-1])      # n
print(text[0:3])     # Pyt
print(len(text))     # 6
print(text.lower())  # python
print(text.upper())  # PYTHON
```

字符串拼接：

```python
first = "Hello"
second = "Python"
print(first + ", " + second)
```

## 布尔值

布尔值只有两个：

```python
True
False
```

常用于判断：

```python
age = 18
print(age >= 18)  # True
```

## 列表 list

列表可以保存多个值，并且可以修改。

```python
fruits = ["apple", "banana", "orange"]
```

常见操作：

```python
print(fruits[0])
fruits.append("pear")
fruits.remove("banana")
print(len(fruits))
```

遍历列表：

```python
for fruit in fruits:
    print(fruit)
```

## 元组 tuple

元组和列表类似，但创建后不能修改。

```python
point = (10, 20)
print(point[0])
```

适合保存不应该被修改的数据，例如坐标、固定配置。

## 字典 dict

字典用键值对保存数据。

```python
user = {
    "name": "Alice",
    "age": 18,
    "city": "Shanghai"
}
```

读取和修改：

```python
print(user["name"])
user["age"] = 19
user["email"] = "alice@example.com"
```

遍历：

```python
for key, value in user.items():
    print(key, value)
```

## 集合 set

集合用于保存不重复的值。

```python
numbers = {1, 2, 3, 3}
print(numbers)  # {1, 2, 3}
```

常用于去重：

```python
names = ["Alice", "Bob", "Alice"]
unique_names = set(names)
print(unique_names)
```

## 类型查看

```python
print(type(123))
print(type("hello"))
print(type([1, 2, 3]))
```

## 怎么选择

| 需求 | 类型 |
| :--- | :--- |
| 保存一个数字 | `int` / `float` |
| 保存文本 | `str` |
| 保存真假 | `bool` |
| 保存一组可修改数据 | `list` |
| 保存一组固定数据 | `tuple` |
| 保存键值对 | `dict` |
| 去重或集合运算 | `set` |
