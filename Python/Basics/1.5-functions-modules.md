# 1.5 函数与模块

函数可以把一段代码封装起来，方便重复使用。模块可以把多个函数放到一个文件里，方便组织项目。

## 定义函数

```python
def say_hello():
    print("Hello")
```

调用函数：

```python
say_hello()
```

## 带参数的函数

```python
def greet(name):
    print(f"你好，{name}")

greet("Alice")
greet("Bob")
```

参数就是函数运行时需要的输入。

## 返回值

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```

`return` 会把结果返回给调用者。

## 默认参数

```python
def greet(name, message="你好"):
    print(f"{message}，{name}")

greet("Alice")
greet("Bob", "早上好")
```

## 函数为什么重要

如果没有函数，代码容易变成一长串，难以维护。

函数可以让代码：

- 更清晰
- 更容易复用
- 更容易测试
- 更容易修改

## 变量作用域

函数内部定义的变量，通常只能在函数内部使用。

```python
def demo():
    x = 10
    print(x)

demo()
# print(x)  # 这里会报错，因为 x 是函数内部变量
```

## 导入模块

Python 有很多内置模块。

```python
import math

print(math.sqrt(16))
print(math.pi)
```

也可以只导入某个函数：

```python
from math import sqrt

print(sqrt(16))
```

## 常用内置模块

| 模块 | 用途 |
| :--- | :--- |
| `os` | 文件路径、目录操作 |
| `sys` | Python 运行环境 |
| `math` | 数学函数 |
| `random` | 随机数 |
| `datetime` | 日期时间 |
| `json` | JSON 数据 |
| `csv` | CSV 文件 |
| `pathlib` | 路径处理 |

## 自己写模块

新建 `tools.py`：

```python
def add(a, b):
    return a + b
```

在 `main.py` 中使用：

```python
import tools

print(tools.add(1, 2))
```

或者：

```python
from tools import add

print(add(1, 2))
```

## main 入口

常见写法：

```python
def main():
    print("程序开始")

if __name__ == "__main__":
    main()
```

含义：只有直接运行当前文件时，才执行 `main()`。

这能让文件既可以作为脚本运行，也可以被其他文件导入。
