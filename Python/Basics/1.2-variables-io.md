# 1.2 变量、注释与输入输出

这一章学习 Python 程序最基础的组成：变量、注释、打印输出和用户输入。

## 变量是什么

变量可以理解为“给数据起一个名字”。

```python
name = "LuoPo"
age = 18
height = 1.75
```

这里：

- `name` 保存字符串
- `age` 保存整数
- `height` 保存小数

## 变量命名规则

可以：

```python
user_name = "Alice"
score = 95
total_price = 19.9
```

不建议或不允许：

```python
2name = "Alice"      # 不能以数字开头
user-name = "Alice"  # 中横线会被当成减号
class = "A"          # class 是 Python 关键字
```

建议：

- 使用英文
- 使用小写字母和下划线
- 名字表达含义

## 注释

注释是写给人看的说明，Python 不会执行。

```python
# 这是一行注释
name = "Alice"  # 这里保存用户名
```

多行说明可以用多个 `#`：

```python
# 计算商品总价
# price 是单价
# count 是数量
price = 10
count = 3
total = price * count
```

## 输出 print

`print()` 用来把内容显示到屏幕上。

```python
print("Hello")
print(123)
print("age =", 18)
```

可以同时打印多个值：

```python
name = "Alice"
age = 18
print("姓名:", name, "年龄:", age)
```

## f-string 格式化输出

推荐使用 f-string：

```python
name = "Alice"
age = 18
print(f"姓名: {name}, 年龄: {age}")
```

`{}` 里可以放变量或表达式：

```python
price = 10
count = 3
print(f"总价: {price * count}")
```

## 输入 input

`input()` 用来读取用户输入。

```python
name = input("请输入你的名字: ")
print(f"你好，{name}")
```

注意：`input()` 得到的内容永远是字符串。

```python
age = input("请输入年龄: ")
print(type(age))
```

如果要做数学计算，需要转换类型：

```python
age = int(input("请输入年龄: "))
print(age + 1)
```

## 常见类型转换

```python
int("123")       # 转成整数
float("3.14")    # 转成小数
str(100)         # 转成字符串
bool(1)          # 转成布尔值
```

1. 输入姓名

3. 输出一句话：`你好，某某，你明年几岁`

```python
name = input("请输入姓名: ")
age = int(input("请输入年龄: "))

```
