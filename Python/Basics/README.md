# Python 基础教程

这是一套写给 Python 新手的入门教程。它不默认你有编程基础，会从“怎么运行第一行代码”开始，逐步讲到变量、数据类型、流程控制、函数、模块、文件读写、异常处理和虚拟环境。

## 适合谁

- 从来没系统学过 Python 的新手
- 会复制脚本运行，但看不懂脚本结构的人
- 想先打好基础，再看 `Auto_scripts` 自动化脚本的人
- 想从 Matlab、Shell 或其他语言转到 Python 的人

## 学习建议

Python 不适合只看不练。建议你每学一节，就新建一个 `.py` 文件，把示例代码敲一遍。

推荐目录：

```text
python-practice/
├── hello.py
├── variables.py
├── lists.py
├── loops.py
├── functions.py
└── files.py
```
