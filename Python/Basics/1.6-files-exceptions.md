# 1.6 文件读写与异常处理

Python 很常用于处理文件，比如读取文本、整理数据、生成报告。

## 读取文本文件

假设有文件 `hello.txt`：

```text
Hello
Python
```

读取文件：

```python
with open("hello.txt", "r", encoding="utf-8") as f:
    content = f.read()

print(content)
```

推荐使用 `with open(...)`，它会自动关闭文件。

## 按行读取

```python
with open("hello.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

`strip()` 会去掉行首行尾的空白和换行符。

## 写入文件

```python
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello\n")
    f.write("Python\n")
```

注意：`w` 模式会覆盖原文件。

## 追加写入

```python
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("New line\n")
```

`a` 表示 append，不会清空原内容。

## 常见文件模式

| 模式 | 含义 |
| :--- | :--- |
| `r` | 读取 |
| `w` | 写入，会覆盖 |
| `a` | 追加 |
| `rb` | 二进制读取 |
| `wb` | 二进制写入 |

## 路径处理

推荐使用 `pathlib`：

```python
from pathlib import Path

file_path = Path("data") / "hello.txt"
print(file_path)
```

判断文件是否存在：

```python
from pathlib import Path

path = Path("hello.txt")

if path.exists():
    print("文件存在")
else:
    print("文件不存在")
```

## 异常是什么

异常就是程序运行时出现的问题。

例如文件不存在：

```python
with open("not_exists.txt", "r", encoding="utf-8") as f:
    print(f.read())
```

可能报错：

```text
FileNotFoundError
```

## try except

用 `try except` 捕获异常：

```python
try:
    with open("not_exists.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("文件不存在，请检查路径")
```

## 捕获多个异常

```python
try:
    number = int(input("请输入数字: "))
    print(10 / number)
except ValueError:
    print("输入的不是数字")
except ZeroDivisionError:
    print("不能除以 0")
```

## 不要滥用 except

不建议一上来就写：

```python
try:
    ...
except:
    pass
```

这样会把真正的问题藏起来，调试时很痛苦。

## 小练习

写一个程序：

1. 读取 `names.txt`
2. 每行一个名字
3. 输出 `你好，名字`
4. 如果文件不存在，提示用户创建文件

参考：

```python
from pathlib import Path

path = Path("names.txt")

if not path.exists():
    print("names.txt 不存在，请先创建文件")
else:
    with open(path, "r", encoding="utf-8") as f:
        for line in f:
            name = line.strip()
            if name:
                print(f"你好，{name}")
```
