# 1.8 新手练习路线

学 Python 最重要的是练习。下面是一条从简单到实用的练习路线。

## 阶段一：基础语法

目标：熟悉变量、输入输出、条件判断和循环。

练习：

1. 输入两个数字，输出和、差、积、商。
2. 输入分数，判断优秀、良好、及格、不及格。
3. 输出 1 到 100 的偶数。
4. 计算 1 到 100 的和。
5. 猜数字小游戏。

示例：猜数字

```python
import random

answer = random.randint(1, 100)

while True:
    guess = int(input("请输入 1 到 100 的数字: "))

    if guess > answer:
        print("大了")
    elif guess < answer:
        print("小了")
    else:
        print("猜对了")
        break
```

## 阶段二：数据结构

目标：熟悉列表、字典、集合。

练习：

1. 统计列表中的最大值、最小值、平均值。
2. 去除列表中的重复元素。
3. 用字典保存学生成绩。
4. 统计一段文字中每个单词出现的次数。

示例：统计单词

```python
text = "python is good python is easy"
words = text.split()

count = {}

for word in words:
    count[word] = count.get(word, 0) + 1

print(count)
```

## 阶段三：函数和模块

目标：把重复代码封装成函数。

练习：

1. 写一个函数判断数字是否为偶数。
2. 写一个函数计算列表平均值。
3. 写一个模块 `tools.py`，放常用函数。
4. 用 `main()` 组织程序入口。

示例：

```python
def average(numbers):
    if not numbers:
        return 0
    return sum(numbers) / len(numbers)

def main():
    scores = [80, 90, 75]
    print(average(scores))

if __name__ == "__main__":
    main()
```

## 阶段四：文件处理

目标：能读取和生成文件。

练习：

1. 读取一个文本文件，统计行数。
2. 读取名单文件，为每个人生成问候语。
3. 批量重命名测试目录里的文件。
4. 把结果写入 `output.txt`。

这个阶段可以开始看本仓库的 [Auto_scripts](../Auto_scripts/README.md)。

## 阶段五：第三方库

目标：学会安装和使用常用库。

推荐库：

| 库 | 用途 |
| :--- | :--- |

| `openpyxl` | Excel 文件 |
| `Pillow` | 图片处理 |

练习：

1. 用 `requests` 请求一个网页。
2. 用 `pandas` 读取 Excel。
3. 用 `Pillow` 调整图片大小。

## 学习习惯

- 先写能跑的版本，再慢慢优化。
- 报错时先读最后一行。
- 每次只改一点，改完就运行。
- 变量名写清楚。
- 不懂的代码加注释。
- 批量操作文件前先在测试目录试。
