# Python 神经网络模型选择指南

这份指南帮助你判断：遇到一个任务时，应该先从哪个神经网络或机器学习模型开始。

这里的建议偏新手友好，不追求一步到位。先用简单模型跑通流程，再逐步换复杂模型。

## 先判断任务类型

| 任务 | 目标 | 推荐先看 |
| :--- | :--- | :--- |
| 回归 | 预测连续数值，例如价格、温度、评分 | [线性回归梯度下降](examples/linear_regression_gradient_descent.py) |
| 二分类 | 判断是/否，例如是否合格、是否故障 | [感知机二分类](examples/perceptron_binary_classification.py)、[逻辑回归二分类](examples/logistic_regression_binary_classification.py) |
| 非线性分类 | 简单直线分不开，例如 XOR | [简单 MLP 学 XOR](examples/simple_mlp_xor.py) |
| 图像分类 | 识别图片类别 | 后续可扩展 CNN 示例 |
| 时间序列 | 根据历史数据预测未来 | 后续可扩展 RNN / LSTM 示例 |

## 新手推荐路线

1. 先学线性回归：理解损失、梯度、参数更新。
2. 再学感知机：理解线性分类和激活函数。
3. 再学逻辑回归：理解概率输出和二分类。
4. 再学 MLP：理解隐藏层为什么有用。
5. 最后再进入 NumPy、scikit-learn、PyTorch。

## 常见模型怎么选

### 线性回归

适合：

- 预测连续数值。
- 输入和输出大致呈线性关系。
- 想学习梯度下降基础。

不适合：

- 类别判断。
- 强非线性关系。

### 感知机

适合：

- 线性可分的二分类。
- 学习“神经元”的最小结构。

不适合：

- XOR 这类非线性问题。
- 需要概率输出的任务。

### 逻辑回归

适合：

- 二分类任务。
- 希望输出接近概率的数值。
- 作为神经网络前的过渡模型。

说明：

逻辑回归名字里有“回归”，但常用于分类。它通过 Sigmoid 函数把线性输出压到 0 到 1 之间。

### 多层感知机

适合：

- 非线性分类。
- 学习隐藏层和反向传播。
- 理解深度学习的基本训练流程。

不适合：

- 很大的图像、语音、文本任务。那些任务通常需要专门结构，比如 CNN、RNN、Transformer。

## 什么时候使用第三方库

当前目录的示例尽量使用标准库，是为了讲清楚原理。

当你开始处理真实数据时，可以考虑：

- `NumPy`：矩阵计算更快。
- `pandas`：读取表格和整理数据更方便。
- `scikit-learn`：传统机器学习模型和评估工具。
- `PyTorch`：深度学习模型训练。

建议不要跳过手写示例。先知道模型内部大概发生了什么，再用框架会更稳。
