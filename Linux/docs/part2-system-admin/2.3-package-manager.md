# 2.3 软件包管理

不同 Linux 发行版使用不同的软件包管理器。

## Debian / Ubuntu：apt

```bash
sudo apt update
sudo apt upgrade
sudo apt install package-name
sudo apt remove package-name
sudo apt autoremove
```

- `apt update`: 更新软件源索引。
- `apt upgrade`: 升级已安装软件。
- `apt install`: 安装软件。
- `apt remove`: 卸载软件。
- `apt autoremove`: 清理不再需要的依赖。

## CentOS：yum

```bash
sudo yum update
sudo yum install package-name
sudo yum remove package-name
sudo yum search keyword
```

`yum` 常见于 CentOS 7 等系统。

## Fedora / 新版 RHEL 系：dnf

```bash
sudo dnf update
sudo dnf install package-name
sudo dnf remove package-name
sudo dnf search keyword
```

`dnf` 可以理解为较新的 `yum`。

## Arch Linux：pacman

```bash
sudo pacman -Syu
sudo pacman -S package-name
sudo pacman -R package-name
sudo pacman -Ss keyword
```

## 安装前建议

安装软件前先确认：

- 当前系统发行版是什么？
- 软件包名称是否正确？
- 是否需要 root 权限？
- 是否会修改系统服务或防火墙？

查看系统：

```bash
cat /etc/os-release
```
