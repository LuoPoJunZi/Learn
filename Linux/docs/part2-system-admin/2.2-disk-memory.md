# 2.2 磁盘、内存和系统信息

服务器排障时，经常要先确认系统版本、运行时间、磁盘和内存是否正常。

## 系统信息

```bash
uname -a
hostname
whoami
id
uptime
lsb_release -a
```

- `uname -a`: 查看内核和系统架构。
- `hostname`: 查看主机名。
- `whoami`: 查看当前用户。
- `id`: 查看用户 ID 和用户组。
- `uptime`: 查看运行时间和负载。
- `lsb_release -a`: 查看发行版信息，部分系统需要安装。

## 磁盘空间

```bash
df -h
du -sh *
du -sh /var/log
```

- `df -h`: 查看磁盘分区使用情况。
- `du -sh *`: 查看当前目录下各项目大小。
- `du -sh 路径`: 查看指定目录大小。

排查磁盘爆满时，可以先看：

```bash
df -h
du -sh /var/log/*
```

## 内存

```bash
free -h
top
htop
```

- `free -h`: 查看内存和 swap。
- `top` / `htop`: 查看哪些进程占用资源。

## 内核日志

```bash
dmesg
dmesg | tail
```

`dmesg` 常用于查看系统启动、硬件、网络、磁盘等底层日志。
