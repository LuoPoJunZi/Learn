# 1.2 文件和目录操作

这一章整理最常用的文件和目录命令。

## 查看目录

```bash
ls
ls -l
ls -a
ls -lh
```

- `ls`: 列出当前目录内容。
- `ls -l`: 显示权限、所有者、大小、修改时间。
- `ls -a`: 显示隐藏文件。
- `ls -lh`: 用更容易读的单位显示文件大小。

## 切换目录

```bash
cd /home/user
cd ..
cd ~
pwd
```

- `cd 路径`: 进入指定目录。
- `cd ..`: 返回上一级目录。
- `cd ~`: 回到当前用户主目录。
- `pwd`: 显示当前所在目录。

## 创建目录和文件

```bash
mkdir new_folder
mkdir -p dir1/dir2/dir3
touch file.txt
```

- `mkdir`: 创建目录。
- `mkdir -p`: 递归创建多级目录。
- `touch`: 创建空文件，或更新时间戳。

## 复制和移动

```bash
cp file1.txt file2.txt
cp -r dir1 dir2
mv old.txt new.txt
mv file.txt /path/to/dir/
```

- `cp`: 复制文件。
- `cp -r`: 复制目录。
- `mv`: 移动或重命名。

## 删除文件和目录

```bash
rm file.txt
rmdir empty_folder
rm -r directory
```

`rm` 删除后通常不会进入回收站。执行前建议先 `ls` 确认路径。

高风险命令：

```bash
rm -rf directory
```

`rm -rf` 会强制递归删除目录，不会询问确认。不要复制网上命令后直接执行，尤其不要在 `/`、`/home`、`/etc` 这类目录下随手运行。

## 查找文件

```bash
find /path -name "*.txt"
find /path -type d -name "backup"
```

- `-name`: 按名称查找。
- `-type d`: 只查找目录。
- `-type f`: 只查找文件。

## 常用压缩与解压

```bash
tar -czvf archive.tar.gz folder/
tar -xzvf archive.tar.gz
zip -r archive.zip folder/
unzip archive.zip
```
