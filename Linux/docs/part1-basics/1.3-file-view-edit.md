# 1.3 文件查看、编辑与搜索

服务器上经常需要查看日志、修改配置、搜索关键词。这一章整理相关命令。

## 查看文件内容

```bash
cat file.txt
less file.txt
more file.txt
head file.txt
head -n 20 file.txt
tail file.txt
tail -n 20 file.txt
tail -f log.txt
```

- `cat`: 直接输出整个文件，适合短文件。
- `less`: 分页查看，适合长文件。
- `head`: 看开头。
- `tail`: 看结尾。
- `tail -f`: 实时查看日志新增内容。

## 编辑文件

### nano

```bash
nano /etc/example.conf
```

`nano` 对新手更友好。底部会显示快捷键，例如 `Ctrl + O` 保存，`Ctrl + X` 退出。

### vim

```bash
vim file.txt
```

常用操作：

```text
i       进入插入模式
Esc     回到普通模式
:w      保存
:q      退出
:wq     保存并退出
:q!     放弃修改并退出
```

## 搜索文本

```bash
grep "keyword" file.txt
grep -r "keyword" /path
grep -n "keyword" file.txt
```

- `grep`: 搜索关键词。
- `grep -r`: 递归搜索目录。
- `grep -n`: 显示行号。

## 文本处理

```bash
awk '{print $1}' file.txt
sed 's/old/new/g' file.txt
```

- `awk`: 常用于按列处理文本。
- `sed`: 常用于替换文本。

新手建议先用 `grep` 和编辑器完成大部分工作，等熟悉后再深入 `awk` 和 `sed`。
