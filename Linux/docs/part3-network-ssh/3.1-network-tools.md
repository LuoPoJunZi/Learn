# 3.1 网络工具与远程传输

网络命令常用于判断服务器是否能访问外网、端口是否通、文件是否能下载。

## ping

```bash
ping example.com
ping 8.8.8.8
```

`ping` 用来测试网络连通性。部分服务器或网站会禁用 ICMP，所以 ping 不通不一定代表服务不可用。

## curl 和 wget

```bash
curl https://example.com
curl -I https://example.com
wget https://example.com/file.tar.gz
```

- `curl`: 常用于请求接口、查看网页响应、下载脚本。
- `curl -I`: 只查看响应头。
- `wget`: 常用于下载文件。

执行远程脚本前，建议先下载查看：

```bash
curl -fsSL https://example.com/install.sh -o install.sh
less install.sh
bash install.sh
```

## 查看 IP

```bash
curl ip.sb
curl ifconfig.me
```

这些命令可以查看当前服务器的出口 IP。

## SSH 登录

```bash
ssh user@server_ip
ssh -p 2222 user@server_ip
```

- `user`: 服务器用户名。
- `server_ip`: 服务器 IP。
- `-p`: 指定 SSH 端口。

## scp 传输文件

```bash
scp file.txt user@server_ip:/home/user/
scp -P 2222 file.txt user@server_ip:/home/user/
scp -r folder user@server_ip:/home/user/
```

注意：`ssh` 使用小写 `-p` 指定端口，`scp` 使用大写 `-P` 指定端口。

## ifconfig 与 ip

老教程常见：

```bash
ifconfig
```

新系统更推荐：

```bash
ip addr
ip route
```
