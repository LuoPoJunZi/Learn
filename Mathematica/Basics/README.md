## Mathematica 的几个核心特点

- 几乎一切都是表达式。
- 函数调用使用方括号：`Sin[x]`，不是 `sin(x)`。
- 列表使用花括号：`{1, 2, 3}`。
- 替换规则使用箭头：`x -> 2`。
- 精确计算和数值计算是两种不同模式。
