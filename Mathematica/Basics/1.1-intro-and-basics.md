# 1.1 Mathematica 是什么与基本操作

Mathematica 是 Wolfram Research 开发的数学计算软件，它使用的语言叫 Wolfram Language。它特别适合做符号计算、数学推导、数值计算、可视化和快速实验。

## Mathematica 适合做什么

- 解方程
- 化简表达式
- 求导和积分
- 矩阵运算
- 绘制函数图像
- 数值求解
- 数据分析和可视化
- 交互式数学实验

## Notebook 是什么

Mathematica 常用 `.nb` Notebook 文件。你可以把它理解成“可运行的数学笔记本”。

Notebook 里可以混合：

- 文本说明
- 数学公式
- 代码
- 运行结果
- 图像

## 最基本的输入输出

在一个输入单元中输入：

```mathematica
1 + 2
```

运行后会得到：

```text
3
```

常见快捷键：

| 操作 | 快捷键 |
| :--- | :--- |
| 运行当前单元 | `Shift + Enter` |
| 新建输入单元 | 点击空白处或在单元后继续输入 |
| 中断计算 | 菜单中的 Abort Evaluation |

## 基本算术运算

```mathematica
3 + 5
7 - 2
4 * 6
10 / 2
2^3
```

说明：

- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `^` 幂运算

## 精确数和近似数

Mathematica 默认倾向于精确计算。

```mathematica
1/3
```

结果是精确分数：

```text
1/3
```

如果写成小数：

```mathematica
1.0/3
```

结果会是近似数。

也可以使用 `N` 转换为近似数：

```mathematica
N[1/3]
N[Pi]
N[Pi, 20]
```

## 常用常量

```mathematica
Pi
E
I
Infinity
```

注意大小写。Mathematica 内置符号通常首字母大写。

## 清除变量

```mathematica
Clear[x]
ClearAll["Global`*"]
```

`Clear[x]` 清除变量 `x` 的值。

`ClearAll["Global`*"]` 常用于清空当前 Notebook 中自己定义的变量。

## 新手最容易犯的错误

### 用小括号调用函数

错误：

```mathematica
Sin(x)
```

正确：

```mathematica
Sin[x]
```

### 忘记大小写

错误：

```mathematica
sin[x]
```

正确：

```mathematica
Sin[x]
```

### 把等号用错

赋值使用：

```mathematica
x = 3
```

方程使用：

```mathematica
x^2 == 4
```

一个等号是赋值，两个等号是判断或方程。
