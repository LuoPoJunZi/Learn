# 1.7 绘图与可视化

Mathematica 的绘图能力很强，适合快速观察函数、数据和数值解。

## 二维函数图像 Plot

```mathematica
Plot[Sin[x], {x, 0, 2 Pi}]
```

多个函数：

```mathematica
Plot[{Sin[x], Cos[x]}, {x, 0, 2 Pi}]
```

## 设置样式

```mathematica
Plot[
    Sin[x],
    {x, 0, 2 Pi},
    PlotStyle -> Red,
    PlotLabel -> "Sine Function",
    AxesLabel -> {"x", "y"}
]
```

## 散点图 ListPlot

```mathematica
data = {{1, 2}, {2, 3}, {3, 5}, {4, 7}}
ListPlot[data]
```

连接点：

```mathematica
ListLinePlot[data]
```

## 三维图 Plot3D

```mathematica
Plot3D[
    Sin[x] Sin[y],
    {x, 0, Pi},
    {y, 0, Pi}
]
```

## 等高线图 ContourPlot

```mathematica
ContourPlot[
    Sin[x] Sin[y],
    {x, 0, 2 Pi},
    {y, 0, 2 Pi}
]
```

## 参数曲线

```mathematica
ParametricPlot[
    {Cos[t], Sin[t]},
    {t, 0, 2 Pi}
]
```

## 直方图

```mathematica
data = RandomVariate[NormalDistribution[0, 1], 1000]
Histogram[data]
```

## 导出图片

```mathematica
plot = Plot[Sin[x], {x, 0, 2 Pi}]
Export["sine.png", plot]
```

常见格式：

```text
png
jpg
pdf
svg
```

## 绘图常用选项

| 选项 | 用途 |
| :--- | :--- |
| `PlotStyle` | 设置线条样式 |
| `PlotLabel` | 设置标题 |
| `AxesLabel` | 设置坐标轴标签 |
| `PlotRange` | 设置显示范围 |
| `ImageSize` | 设置图片大小 |
| `GridLines` | 显示网格线 |
| `Legends` | 设置图例 |

## 新手建议

先把图画出来，再慢慢加样式。

比如先写：

```mathematica
Plot[Sin[x], {x, 0, 2 Pi}]
```

确认图像正确后，再添加标题、颜色、标签。
