# 1.8 文件导入导出与练习路线

这一章介绍基础的导入导出，并给出一条练习路线。

## 导入数据 Import

```mathematica
data = Import["data.csv"]
```

导入 Excel：

```mathematica
data = Import["data.xlsx"]
```

导入图片：

```mathematica
img = Import["image.png"]
```

## 导出数据 Export

```mathematica
Export["output.csv", data]
Export["plot.png", plot]
```

常见格式：

```text
csv
xlsx
txt
png
pdf
svg
```

## 查看当前目录

```mathematica
Directory[]
```

设置当前目录：

```mathematica
SetDirectory["D:\\data"]
```

建议把数据文件和 Notebook 放在同一个项目文件夹里，路径更不容易乱。

## 保存和读取 Mathematica 数据

```mathematica
DumpSave["vars.mx", {x, y, data}]
```

读取：

```mathematica
Get["vars.mx"]
```

普通新手项目中，更多时候使用 `Export` 和 `Import` 即可。

## 阶段一：基础表达式

练习：

1. 计算 `3 + 5 * 2`
2. 计算 `2^10`
3. 用 `N[Pi, 20]` 输出 20 位圆周率
4. 比较 `1/3` 和 `1.0/3` 的区别

## 阶段二：符号计算

练习：

```mathematica
Simplify[Sin[x]^2 + Cos[x]^2]
Expand[(x + 2)^3]
Factor[x^2 - 5 x + 6]
Solve[x^2 - 4 == 0, x]
```

## 阶段三：列表和函数

练习：

```mathematica
Table[i^2, {i, 1, 10}]
Select[Range[1, 20], EvenQ]
Map[#^2 &, Range[1, 5]]
Total[Range[1, 100]]
```

## 阶段四：矩阵和微积分

练习：

```mathematica
A = {{1, 2}, {3, 4}}
Det[A]
Inverse[A]
Eigenvalues[A]
D[x^3 + 2 x, x]
Integrate[x^2, {x, 0, 1}]
```

## 阶段五：绘图

练习：

```mathematica
Plot[Sin[x], {x, 0, 2 Pi}]
Plot[{Sin[x], Cos[x]}, {x, 0, 2 Pi}]
Plot3D[Sin[x] Sin[y], {x, 0, Pi}, {y, 0, Pi}]
```

## 阶段六：小项目

可以尝试：

- 解一个方程组并可视化结果
- 绘制常见函数图像合集
- 读取 CSV 数据并画散点图
- 用 `NDSolve` 解简单微分方程并绘图

## 学习习惯

- 注意函数名大小写。
- 函数调用使用 `[]`。
- 方程使用 `==`，赋值使用 `=`。
- 列表索引从 1 开始。
- 不确定表达式结构时，用 `FullForm` 看内部形式。
- 复杂计算前先用简单例子验证。
