# 1.3 列表、规则与常用操作

列表是 Mathematica 中最常用的数据结构之一。向量、矩阵、数据表都可以用列表表示。

## 创建列表

```mathematica
list = {1, 2, 3, 4, 5}
```

字符串列表：

```mathematica
names = {"Alice", "Bob", "Charlie"}
```

## 访问列表元素

Mathematica 的列表索引从 1 开始。

```mathematica
list[[1]]
list[[2]]
list[[-1]]
```

- `list[[1]]` 第一个元素
- `list[[-1]]` 最后一个元素

## 生成序列

```mathematica
Range[1, 5]
Range[1, 10, 2]
Table[i^2, {i, 1, 5}]
```

结果示例：

```text
{1, 2, 3, 4, 5}
{1, 3, 5, 7, 9}
{1, 4, 9, 16, 25}
```

## 列表常用函数

```mathematica
Length[list]
First[list]
Last[list]
Rest[list]
Take[list, 3]
Drop[list, 2]
Reverse[list]
Sort[list]
```

## Map

`Map` 把函数应用到列表的每个元素。

```mathematica
Map[Sin, {0, Pi/2, Pi}]
```

也可以写成简写：

```mathematica
Sin /@ {0, Pi/2, Pi}
```

## Apply

`Apply` 会把列表头部替换成指定函数。

```mathematica
Apply[Plus, {1, 2, 3}]
```

简写：

```mathematica
Plus @@ {1, 2, 3}
```

结果：

```text
6
```

## Select

筛选列表：

```mathematica
Select[{1, 2, 3, 4, 5}, # > 3 &]
```

结果：

```text
{4, 5}
```

## 规则

规则使用 `->`：

```mathematica
x -> 2
```

替换规则：

```mathematica
x^2 + y /. {x -> 3, y -> 4}
```

## Association

Association 类似其他语言中的字典。

```mathematica
user = <|"name" -> "Alice", "age" -> 18|>
```

读取：

```mathematica
user["name"]
user["age"]
```

## 列表和矩阵

二维列表可以表示矩阵：

```mathematica
matrix = {{1, 2}, {3, 4}}
```

访问第 1 行第 2 列：

```mathematica
matrix[[1, 2]]
```
