# 1.6 微积分与数值计算

Mathematica 可以做符号微积分，也可以做数值求解。

## 求导

```mathematica
D[x^2 + 3 x + 1, x]
```

结果：

```text
3 + 2 x
```

高阶导数：

```mathematica
D[Sin[x], {x, 2}]
```

偏导：

```mathematica
D[x^2 y + y^2, x]
D[x^2 y + y^2, y]
```

## 积分

不定积分：

```mathematica
Integrate[x^2, x]
```

定积分：

```mathematica
Integrate[x^2, {x, 0, 1}]
```

多重积分：

```mathematica
Integrate[x y, {x, 0, 1}, {y, 0, 1}]
```

## 极限

```mathematica
Limit[Sin[x]/x, x -> 0]
```

左右极限：

```mathematica
Limit[1/x, x -> 0, Direction -> "FromAbove"]
Limit[1/x, x -> 0, Direction -> "FromBelow"]
```

## 数值化 N

```mathematica
N[Pi]
N[Pi, 30]
```

`N[expr, n]` 可以指定有效数字。

## 数值求根 FindRoot

```mathematica
FindRoot[Cos[x] - x == 0, {x, 0}]
```

`{x, 0}` 表示变量是 `x`，初始值为 0。

数值方法依赖初值，不同初值可能得到不同结果。

## 数值解方程 NSolve

```mathematica
NSolve[x^3 - x == 0, x]
```

`NSolve` 返回数值解。

符号解可以用：

```mathematica
Solve[x^3 - x == 0, x]
```

## 数值积分 NIntegrate

```mathematica
NIntegrate[Exp[-x^2], {x, 0, 1}]
```

当符号积分很难或没有解析解时，使用 `NIntegrate`。

## 数值微分方程 NDSolve

```mathematica
sol = NDSolve[
    {y'[x] == y[x], y[0] == 1},
    y,
    {x, 0, 5}
]
```

绘图：

```mathematica
Plot[Evaluate[y[x] /. sol], {x, 0, 5}]
```

## 精确计算和数值计算怎么选

| 需求 | 推荐 |
| :--- | :--- |
| 想得到公式形式 | `Solve`、`Integrate`、`DSolve` |
| 想得到小数结果 | `NSolve`、`NIntegrate`、`NDSolve` |
| 表达式太复杂 | 先尝试数值方法 |
| 需要高精度 | 使用 `N[expr, digits]` |
