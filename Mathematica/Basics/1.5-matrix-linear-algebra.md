# 1.5 矩阵与线性代数

Mathematica 使用嵌套列表表示矩阵。

## 创建矩阵

```mathematica
A = {{1, 2}, {3, 4}}
```

显示为矩阵形式：

```mathematica
MatrixForm[A]
```

注意：`MatrixForm` 主要用于显示，不建议把 `MatrixForm[A]` 赋值给变量继续计算。

## 访问元素

```mathematica
A[[1, 1]]
A[[1, 2]]
A[[2, 1]]
```

Mathematica 索引从 1 开始。

## 矩阵加法和乘法

```mathematica
B = {{5, 6}, {7, 8}}

A + B
A . B
```

注意：矩阵乘法使用点号 `.`。

逐元素乘法：

```mathematica
A * B
```

## 转置

```mathematica
Transpose[A]
```

## 行列式

```mathematica
Det[A]
```

## 逆矩阵

```mathematica
Inverse[A]
```

如果矩阵不可逆，会出现错误或警告。

## 特征值和特征向量

```mathematica
Eigenvalues[A]
Eigenvectors[A]
Eigensystem[A]
```

`Eigensystem` 会同时返回特征值和特征向量。

## 单位矩阵和零矩阵

```mathematica
IdentityMatrix[3]
ConstantArray[0, {3, 3}]
ConstantArray[1, {2, 4}]
```

## 解线性方程组

例如：

```text
x + 2 y = 5
3 x + 4 y = 11
```

可以写成矩阵形式：

```mathematica
A = {{1, 2}, {3, 4}}
b = {5, 11}
LinearSolve[A, b]
```

也可以用：

```mathematica
Solve[
    {x + 2 y == 5, 3 x + 4 y == 11},
    {x, y}
]
```

## 常见错误

### 用 `*` 做矩阵乘法

```mathematica
A * B
```

这是逐元素相乘。

矩阵乘法应该用：

```mathematica
A . B
```

### 把 MatrixForm 结果继续计算

不建议：

```mathematica
A = MatrixForm[{{1, 2}, {3, 4}}]
Det[A]
```

建议：

```mathematica
A = {{1, 2}, {3, 4}}
MatrixForm[A]
Det[A]
```
