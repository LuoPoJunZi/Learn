# 1.2 表达式、变量与函数调用

Mathematica 的核心思想是表达式。数字、符号、公式、函数调用、列表，本质上都可以看作表达式。

## 变量赋值

```mathematica
x = 10
y = x + 5
```

运行后：

```mathematica
y
```

会得到：

```text
15
```

## 延迟赋值

Mathematica 有两种常见赋值方式：

```mathematica
=
:=
```

立即赋值：

```mathematica
a = RandomInteger[10]
```

`a` 会立刻得到一个随机数，以后再输入 `a`，值不变。

延迟赋值：

```mathematica
b := RandomInteger[10]
```

每次输入 `b` 都会重新计算。

## 函数调用

Mathematica 使用方括号调用函数：

```mathematica
Sqrt[16]
Log[10]
Sin[Pi/2]
Cos[0]
Exp[1]
```

多个参数用逗号分隔：

```mathematica
Power[2, 3]
Max[1, 5, 3]
Min[1, 5, 3]
```

## 自定义函数

定义平方函数：

```mathematica
square[x_] := x^2
```

调用：

```mathematica
square[5]
```

结果：

```text
25
```

这里的 `x_` 表示一个模式，意思是“任意输入参数”。

## 多参数函数

```mathematica
add[a_, b_] := a + b
add[3, 5]
```

## 匿名函数

```mathematica
(#^2) &[5]
```

结果：

```text
25
```

解释：

- `#` 表示输入参数
- `&` 表示这是一个匿名函数

匿名函数常和 `Map`、`Apply` 一起用。

## 替换变量

使用 `/.` 和规则 `->`：

```mathematica
x^2 + 3 x + 1 /. x -> 2
```

结果：

```text
11
```

多个替换：

```mathematica
x + y /. {x -> 1, y -> 2}
```

## 表达式展开和查看结构

```mathematica
FullForm[x + y]
```

Mathematica 内部会把表达式表示成树状结构。

例如：

```mathematica
x + y
```

可以理解为：

```mathematica
Plus[x, y]
```

这就是为什么 Mathematica 特别适合符号计算。
