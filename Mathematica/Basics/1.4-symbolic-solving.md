# 1.4 符号计算与方程求解

符号计算是 Mathematica 最强的能力之一。它可以对表达式进行化简、展开、因式分解和求解方程。

## 化简 Simplify

```mathematica
Simplify[Sin[x]^2 + Cos[x]^2]
```

结果：

```text
1
```

更强的化简：

```mathematica
FullSimplify[expr]
```

`FullSimplify` 更强，但可能更慢。

## 展开 Expand

```mathematica
Expand[(x + 1)^2]
```

结果：

```text
1 + 2 x + x^2
```

## 因式分解 Factor

```mathematica
Factor[x^2 - 2 x - 3]
```

结果：

```text
(-3 + x) (1 + x)
```

## 收集同类项 Collect

```mathematica
Collect[a x^2 + b x + c + d x, x]
```

## 解方程 Solve

```mathematica
Solve[x^2 - 4 == 0, x]
```

结果：

```text
{{x -> -2}, {x -> 2}}
```

注意方程中使用 `==`。

## 解方程组

```mathematica
Solve[
    {x + y == 3, x - y == 1},
    {x, y}
]
```

结果：

```text
{{x -> 2, y -> 1}}
```

## 使用结果

`Solve` 返回的是规则列表。可以用替换规则取值：

```mathematica
solution = Solve[x^2 - 4 == 0, x]
x /. solution
```

## Reduce

`Reduce` 比 `Solve` 更偏数学描述，适合复杂条件。

```mathematica
Reduce[x^2 - 4 == 0, x]
```

结果类似：

```text
x == -2 || x == 2
```

## 假设条件

```mathematica
Simplify[Sqrt[x^2], x > 0]
```

或者：

```mathematica
FullSimplify[Sqrt[x^2], Assumptions -> x > 0]
```

## 微分方程 DSolve

```mathematica
DSolve[y'[x] == y[x], y[x], x]
```

`DSolve` 用于符号求解微分方程。

如果符号解很难，可以考虑数值方法 `NDSolve`。
