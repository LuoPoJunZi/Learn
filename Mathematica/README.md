## 学习建议

Mathematica 很适合做符号推导、数学实验和快速可视化。新手建议先熟悉：

- 函数调用格式：`Function[argument]`
- 列表格式：`{a, b, c}`
- 替换规则：`x -> value`
- 精确计算和数值计算的区别
