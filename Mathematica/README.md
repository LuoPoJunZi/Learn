## 学习建议

Mathematica 很适合做符号推导、数学实验和快速可视化。新手建议先熟悉：

- 函数调用格式：`Function[argument]`
- 列表格式：`{1, 2, 3}`
- 替换规则：`x -> value`
- 精确计算和数值计算的区别
- 赋值 `=` 和方程 `==` 的区别

如果你只是想快速查某个命令，可以看 [Mathematica.md](Mathematica.md)。

如果你想先建立学习路线、理解规则替换、符号计算、绘图和与 Matlab/Python 的区别，可以先看 [Mathematica 学习导读](LEARNING_GUIDE.md)。
