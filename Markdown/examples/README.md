## 怎么使用模板

建议不要直接复制整篇后不改内容。更好的方式是：

1. 先通读一遍模板标题。
2. 删除当前项目用不到的部分。
3. 把示例命令改成真实命令。
4. 把风险提醒改成当前项目真正需要注意的内容。
5. 检查所有链接是否能打开。

## README 写作检查表

写完 README 后，可以快速检查：

- 标题是否能看出项目用途。
- 第一段是否说明“这是什么”。
- 是否有安装或运行步骤。
- 是否有输入、输出或示例结果。
- 是否标注依赖版本。
- 是否提醒危险操作或敏感信息。
- 仓库内链接是否使用相对路径。
- 代码块是否标注语言。
