# Learn

一个面向新手的学习资料库，用来整理 GitHub、Linux、Markdown、Python、HTML、Matlab、Mathematica 等教程、脚本示例和实践笔记。

这个仓库里的内容一部分来自开源项目和公开资料，一部分是结合 AI 重新整理后的学习笔记。你可以把它当成个人知识库，也可以按目录直接查找需要的教程、命令和示例代码。

## 各目录说明

### GitHub

[Github](Github/README.md) 是一套面向新手的 GitHub 教程，覆盖：

- Git 与 GitHub 的区别
- Git 安装、账号配置、SSH 密钥
- 工作区、暂存区、本地仓库、远程仓库
- 常用 Git 命令
- 本地项目上传到 GitHub
- VS Code 图形化 Git 工作流
- 分支、Pull Request、冲突解决
- `.gitignore`、Commit 规范、Issue 写法
- GitHub 脚本短链和 Cloudflare Workers 实用技巧

### Linux

[Linux](Linux/README.md) 已整理成“基础教程 + 系统管理 + SSH 安全 + VPS 工具”的结构，适合从零理解命令行和服务器操作。

重点内容：

- Linux 入门概念
- 文件和目录操作
- 文件查看、编辑、搜索
- 权限与所有者
- 进程、服务、系统资源
- 软件包管理
- 网络工具与 SSH 登录
- SSH 安全加固和日志排查
- VPS 常用脚本速查表

### Markdown

[Markdown](Markdown/README.md) 是一份完整的 Markdown 语法教程，适合写 GitHub README、博客、教程和学习笔记。

重点内容：

- 标题、段落、换行
- 加粗、斜体、删除线
- 引用、列表、任务列表
- 链接、图片
- 行内代码、代码块
- 表格、分割线、脚注
- 数学公式和 Mermaid 图表
- CSDN、Hexo / 安知鱼主题扩展语法说明

### Python

[Python](Python/README.md) 目录包含 Python 基础教程和脚本示例：

- [优秀开源仓库导读](Python/EXTERNAL_REPOSITORIES.md)：适合初学者继续学习的 Python 外部资源
- [Basics](Python/Basics/README.md)：Python 基础教程，适合新手入门
- [Auto_scripts](Python/Auto_scripts/README.md)：日常自动化脚本集合
- [Neural Network](<Python/Neural Network/README.md>)：神经网络入门示例
- [Multi-Objective Optimization](<Python/Multi-Objective Optimization/README.md>)：多目标优化入门示例
- [Grab](Python/Grab/README.md)：图片抓取示例
- [Love](Python/Love/README.md)：Python 图形小程序示例

自动化脚本涉及文件整理、网站检测、批量重命名、PDF 文本提取、Excel 处理、图片处理、邮件发送等场景。运行前建议先看 [依赖总览](Python/Auto_scripts/DEPENDENCIES.md)。
