# 1.2 账号注册与安装 Git

开始使用 GitHub 前，需要准备两样东西：

- 一个 GitHub 账号
- 本地电脑上的 Git 工具

## 注册 GitHub 账号

1. 打开 GitHub 官网：`https://github.com`
2. 点击 `Sign up`。
3. 填写邮箱、密码和用户名。
4. 根据页面提示完成邮箱验证。

用户名会出现在你的个人主页和仓库地址里，例如：

```text
https://github.com/your-name
```

建议用户名简短、清晰、方便别人识别。

## 安装 Git

### Windows

1. 打开 Git 官网：`https://git-scm.com`
2. 下载 Windows 版本安装包。
3. 一路按默认选项安装即可。
4. 安装完成后，在开始菜单里找到 `Git Bash`。

也可以在 PowerShell 或 VS Code 终端里使用 Git。

### macOS

打开终端，输入：

```bash
git --version
```

如果系统提示安装 Command Line Tools，按提示安装即可。

也可以使用 Homebrew：

```bash
brew install git
```

### Linux

Ubuntu 或 Debian：

```bash
sudo apt update
sudo apt install git
```

CentOS 或 Fedora：

```bash
sudo dnf install git
```

## 检查 Git 是否安装成功

打开终端，输入：

```bash
git --version
```

如果看到类似下面的输出，说明安装成功：

```text
git version 2.x.x
```

## 配置用户名和邮箱

第一次使用 Git，需要告诉 Git “你是谁”。这些信息会写进提交记录里。

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

查看配置：

```bash
git config --global --list
```

邮箱建议使用 GitHub 账号绑定的邮箱。这样 GitHub 才能把你的提交记录正确关联到你的账号。

## 推荐安装 VS Code

VS Code 对 Git 支持很好。新手可以先用 VS Code 的图形界面完成大部分操作，再慢慢理解命令。

下载地址：

```text
https://code.visualstudio.com
```

安装后建议安装这些扩展：

- Chinese Language Pack for Visual Studio Code
- GitLens
- GitHub Pull Requests

## 常见问题

### 输入 git 后提示 command not found

说明 Git 没有安装成功，或者没有加入系统 PATH。重新安装 Git，并勾选允许在命令行中使用 Git。

### GitHub 打不开或很慢

可能是网络问题。可以先学习本地 Git 操作，等网络稳定后再进行 clone、push、pull 等远程操作。

### 用户名邮箱写错了怎么办

重新执行配置命令即可覆盖：

```bash
git config --global user.name "新名字"
git config --global user.email "新邮箱"
```
